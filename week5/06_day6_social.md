## 📖 Phần 3: Bài Đọc Tech + Life (15 phút)

### Bài Đọc 5 — 一个越南程序员在中国 (200 chữ)

```
小阮是越南人，在深圳一家互联网公司做后端开发。
他去年来的中国。来之前，他学了半年中文。

刚来的时候，他的中文不太好。虽然会写代码，
但是不知道怎么用中文和同事交流。每次开会都
很紧张。同事们都说英文太慢，最好用中文。

他决定每天提高自己的中文水平。每天下班以后
学习一个小时。除了学技术中文以外，他还学了
很多关于中国文化的知识。

半年以后，他的中文进步了很多。不但能用中文
开会了，而且还能看懂中国同事写的技术文档。
同事们都对他的进步很满意。

现在小阮觉得来中国工作是一个很好的机会。
虽然工作有时候很紧张，但是周末很轻松，
可以去了解深圳的文化和生活。他丰富的工作
经验和中文能力让他在公司的关系越来越好。

他说："学中文不但提高了我的工作能力，
而且让我交到了很多中国朋友。"
```

## 🗣️ Phần 4: Hội Thoại — Đời Sống Dev (15 phút)

```
A: 你在中国工作紧张吗？
B: 有时候有点紧张，特别是上线的时候。
   但是周末很轻松。

A: 你平时和同事怎么交流？
B: 主要用中文。刚来的时候不太会，但是现在
   已经适合了。学中文给了我很多机会。

A: 你觉得在中国工作需要什么？
B: 不但需要技术好，而且中文也要好。
   了解中国文化也很重要。

A: 你对现在的工作满意吗？
```

## ✏️ Phần 5: 8 Hán Tự Mới (20 phút)

#### 验 yàn (nghiệm) — 10 nét
```
经验 = kinh nghiệm, 实验 = thí nghiệm
Tập viết: 验 验 验 验 验 验 验 验 验 验
```
#### 机 jī (cơ/máy) — 6 nét
```
机会 = cơ hội, 手机 = ĐT, 飞机 = máy bay
Tập viết: 机 机 机 机 机 机 机 机 机 机
```
#### 关 guān (quan/đóng) — 6 nét
```
关系 = quan hệ, 关门 = đóng cửa
Tập viết: 关 关 关 关 关 关 关 关 关 关
```
#### 活 huó (sống/hoạt) — 9 nét
```
活动 = hoạt động, 生活 = cuộc sống
Tập viết: 活 活 活 活 活 活 活 活 活 活
```
#### 流 liú (lưu/chảy) — 10 nét
```
交流 = giao lưu, 流行 = phổ biến
Tập viết: 流 流 流 流 流 流 流 流 流 流
```
#### 适 shì (thích/hợp) — 9 nét
```
适合 = phù hợp, 合适 = phù hợp
Tập viết: 适 适 适 适 适 适 适 适 适 适
```
#### 丰 fēng (phong/giàu) — 4 nét
```
丰富 = phong phú/giàu có
Tập viết: 丰 丰 丰 丰 丰 丰 丰 丰 丰 丰
```
#### 松 sōng (tùng/lỏng) — 8 nét
```
轻松 = thoải mái, 放松 = thư giãn
Tập viết: 松 松 松 松 松 松 松 松 松 松
```
