### API & Backend:

```
请求     qǐngqiú     request         yêu cầu
响应     xiǎngyìng   response        phản hồi
返回     fǎnhuí      return          trả về
传       chuán       truyền/gửi      send/transmit
参数     cānshù      tham số         parameter
```

### Câu thực tế:

```
前端用React，后端用Spring Boot。
Frontend dùng React, backend dùng Spring Boot.

用户点击链接以后，浏览器发送请求到服务器。
User click link, browser gửi request đến server.

服务器处理请求，然后返回数据给前端。
Server xử lý request, rồi return data cho frontend.

这个接口需要两个参数：用户名和密码。
API này cần 2 tham số: username và password.

用户登录以后可以看到自己的页面。
User đăng nhập xong có thể xem trang của mình.
```

> 💡 **Dev tip:** Luồng web TQ cũng giống: 浏览器(browser) → 请求(request) → 服务器(server) → 响应(response) → 页面(page). Đọc docs TQ sẽ thấy các từ này LIÊN TỤC!

## 📖 Phần 3: Bài Đọc Tech (15 phút)

### Bài Đọc 2 — Web开发入门 (180 chữ)

```
Web开发分为前端和后端。前端是用户能看到的
部分，比如页面、按钮和图片。后端是用户看不到
的部分，比如服务器、数据库和接口。

做前端开发要学HTML、CSS和JavaScript。现在最
流行的前端框架是React和Vue。做后端开发要学
Java、Python或者Go。

一个网站是怎么工作的呢？首先，用户在浏览器
里输入网址。浏览器向服务器发送一个请求。
服务器收到请求以后，从数据库里读取数据，
然后把数据返回给浏览器。浏览器收到数据以后，
在页面上显示出来。

除了前端和后端以外，还有全栈开发。全栈开发
就是前端和后端都会做。我就是一个全栈开发，
虽然还不太熟练，但是正在努力学习。
```

## 🗣️ Phần 4: Hội Thoại — Web Dev Discussion (15 phút)

```
A: 你做前端还是后端？
   Nǐ zuò qiánduān háishi hòuduān?

B: 我做后端。你呢？
   Wǒ zuò hòuduān. Nǐ ne?

A: 我做前端，用React。你们的接口用什么？
   Wǒ zuò qiánduān, yòng React. Nǐmen de jiēkǒu yòng shénme?

B: RESTful API。每个接口都有参数说明。
   RESTful API. Měi ge jiēkǒu dōu yǒu cānshù shuōmíng.

A: 用户登录的接口需要什么参数？
   Yònghù dēnglù de jiēkǒu xūyào shénme cānshù?

B: 需要用户名和密码。登录成功以后返回一个token。
   Xūyào yònghùmíng hé mìmǎ. Dēnglù chénggōng yǐhòu fǎnhuí yí ge token.

A: 好的。页面加载有点慢，你能不能看一下后端？
   Hǎo de. Yèmiàn jiāzài yǒudiǎn màn, nǐ néng bu néng kàn yíxià hòuduān?

B: 可能是数据库查询太慢。我看一下。
   Kěnéng shì shùjùkù cháxún tài màn. Wǒ kàn yíxià.
```
