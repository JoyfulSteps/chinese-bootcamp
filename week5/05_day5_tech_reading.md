## 📐 Phần 1: Ngữ Pháp — 不但...而且 (20 phút)

### Pattern: 不但 A 而且 B = Không chỉ A mà còn B

```
不但 búdàn = không chỉ     而且 érqiě = mà còn

不但...而且... luôn đi CÙNG HƯỚNG (tốt+tốt hoặc xấu+xấu)
→ Vế sau MẠNH hơn vế trước
```

### Ví dụ:

```
他不但会写Java，而且还会写Python。
Anh ấy không chỉ biết Java, mà còn biết Python.

这个框架不但好用，而且性能很好。
Framework này không chỉ dễ dùng, mà còn hiệu suất tốt.

学中文不但有用，而且很有意思。
Học TQ không chỉ hữu ích, mà còn rất thú vị.

这个bug不但影响了前端，而且后端也有问题。
Bug này không chỉ ảnh hưởng frontend, mà backend cũng có vấn đề.
```

### Phân biệt 除了 vs 不但:

```
除了Java以外，我还会Python。
    = Ngoài Java, tôi CÒN biết Python. (liệt kê)

不但会Java，而且还会Python。
    = Không CHỈ biết Java, mà CÒN biết Python. (nhấn mạnh)

→ 不但而且 NHẤN MẠNH hơn (wow factor)
→ 除了以外 CHỈ LIỆT KÊ thêm
```

> 💡 **Dev tip:** `除了...以外` = `list.push()` — thêm phần tử bình thường. `不但...而且` = `list.push()` + `highlight()` — thêm VÀ nhấn mạnh: "ẤN TƯỢNG NHÉ!"

## 📖 Phần 2: Guided Reading — Kỹ Thuật Đọc Tech (30 phút)

### 3 Kỹ Thuật Đọc Tech Tiếng Trung:

```
1️⃣ 猜 (cāi) = ĐOÁN từ ngữ cảnh
   → Thấy từ lạ? Đoán từ câu xung quanh + từ Hán-Việt
   → 优化 = ưu hóa = optimize (Hán-Việt!)
   → 文档 = văn bản tài liệu = documentation

2️⃣ 找 (zhǎo) = TÌM từ khóa quen
   → Scan bài: tìm từ quen (代码, 框架, 数据...)
   → Từ quen = ≥ 30% → hiểu ý chính

3️⃣ 跳 (tiào) = NHẢY qua từ không quan trọng
   → Đọc lần 1: chỉ cần hiểu ý chính (60%)
   → Đọc lần 2: tra từ quan trọng
   → Không cần hiểu 100%!
```

### Thực Hành: Đọc GitHub README Mẫu (200 chữ)

```
# Vue.js 简介

Vue.js 是一个用于创建用户界面的JavaScript框架。

## 特点
- 简单易学：Vue的语法简单，容易上手。
  不但初学者能快速学会，而且有经验的开发者
  也很喜欢用它。
- 性能好：Vue使用虚拟DOM技术，页面更新很快。
- 生态丰富：除了Vue本身以外，还有Vue Router、
  Vuex等工具可以一起使用。

## 安装
使用npm安装：
npm install vue

## 快速开始
创建一个新项目：
npm create vue@latest my-app

## 文档
完整的文档请访问官方网站。
如果在使用中遇到问题，可以在GitHub上提交issue。

## 贡献
我们欢迎所有的贡献！请先阅读贡献指南，
然后创建一个分支，提交你的代码。
代码审查通过以后，我们会合并到主分支。
```

## 📖 Phần 4: Bài Đọc Tech Nâng Cao (15 phút)

### Bài Đọc 4 — 选择框架 (190 chữ)

```
现在的前端框架很多，最流行的有React、Vue和
Angular。每个框架都有自己的特点。

React是Facebook开发的。它不但社区很大，
而且生态很丰富。很多大公司都在用React。
但是React的学习曲线比较陡。

Vue是中国人开发的，所以中文文档很好。
Vue不但简单易学，而且性能也不错。
很多中国公司喜欢用Vue。

选择框架的时候要考虑几个因素：
第一，项目的需求是什么？简单的项目可以用Vue，
复杂的项目可能更适合React。
第二，团队成员会什么？如果大家都会React，
就不需要学新的框架。
第三，性能需求怎么样？如果需要很高的性能，
要看每个框架的优化方法。

除了框架以外，工具的选择也很重要。
好的开发工具可以大大提高开发效率。
```
