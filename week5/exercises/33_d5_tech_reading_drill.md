## Bài Tập 1: 不但...而且 (12 phút)

### 1.1 Dịch sang TQ dùng 不但...而且:

| # | Tiếng Việt | Đáp án |
|---|-----------|--------|
| 1 | FW này không chỉ dễ dùng, mà còn nhanh. | <details><summary>🔑</summary>这个框架不但好用，而且（速度）很快。</details> |
| 2 | Anh ấy không chỉ biết Java, mà còn biết Python. | <details><summary>🔑</summary>他不但会Java，而且还会Python。</details> |
| 3 | Bug này kchỉ ảnh hưởng FE, mà BE cũng có vấn đề. | <details><summary>🔑</summary>这个bug不但影响了前端，而且后端也有问题。</details> |
| 4 | Học TQ không chỉ hữu ích, mà còn thú vị. | <details><summary>🔑</summary>学中文不但有用，而且很有意思。</details> |

### 1.2 Kết hợp 除了 vs 不但 — Chọn đúng:

| # | Câu | 除了 hay 不但？ | Đáp án |
|---|-----|---------------|--------|
| 1 | ___Java___，我还会Python和Go。(liệt kê bình thường) | | <details><summary>🔑</summary>除了Java以外</details> |
| 2 | 他___聪明，___很努力。(nhấn mạnh: wow!) | | <details><summary>🔑</summary>不但…而且</details> |
| 3 | ___星期天___，我每天都上班。(trừ ra) | | <details><summary>🔑</summary>除了…以外，都</details> |

## Bài Tập 3: Guided Reading — Đọc README Thực Tế (20 phút)

Đọc đoạn sau (mô phỏng README thực tế), dùng 3 kỹ thuật: 猜找跳

```
# Ant Design

Ant Design 是阿里巴巴开发的企业级UI组件库。

## 主要特性
- 提供丰富的高质量React组件
- 使用TypeScript编写，支持类型检查
- 国际化支持，覆盖数十种语言
- 不但支持浏览器端，而且支持服务端渲染

## 安装方法
npm install antd

## 使用方法
import { Button } from 'antd';
```

| # | 问题 | Đáp án |
|---|------|--------|
| 1 | Ant Design是谁开发的？ | <details><summary>🔑</summary>阿里巴巴 (Alibaba)</details> |
| 2 | 用什么语言写的？ | <details><summary>🔑</summary>TypeScript</details> |
| 3 | 怎么安装？ | <details><summary>🔑</summary>npm install antd</details> |
| 4 | 除了浏览器端，还支持什么？ | <details><summary>🔑</summary>服务端渲染 (server-side rendering)</details> |
