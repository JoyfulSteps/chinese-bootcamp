## Bài Tập 1: 除了...以外 — Phân Biệt 也/还 vs 都 (12 phút)

### 1.1 Chọn 也/还 (thêm) hoặc 都 (trừ):

| # | Câu | 也/还 hay 都？ | Đáp án |
|---|-----|--------------|--------|
| 1 | 除了Java以外，我___会写Python。 | | <details><summary>🔑</summary>**还** (thêm: ngoài Java, CÒN biết Python)</details> |
| 2 | 除了星期天以外，我每天___上班。 | | <details><summary>🔑</summary>**都** (trừ: trừ CN ra, ngày nào CŨNG đi làm)</details> |
| 3 | 除了他以外，同事们___来了。 | | <details><summary>🔑</summary>**都** (trừ: trừ anh ấy, mọi người ĐỀU đến)</details> |
| 4 | 除了写代码以外，我___要测试。 | | <details><summary>🔑</summary>**还** (thêm: ngoài code, CÒN phải test)</details> |
| 5 | 除了这个bug以外，代码___没问题。 | | <details><summary>🔑</summary>**都** (trừ: trừ bug này, đều OK)</details> |

### 1.2 Dịch sang TQ:

| # | Tiếng Việt | Đáp án |
|---|-----------|--------|
| 1 | Ngoài Python ra, tôi còn biết Java. | <details><summary>🔑</summary>除了Python以外，我还会Java。</details> |
| 2 | Trừ anh ấy ra, ai cũng đến. | <details><summary>🔑</summary>除了他以外，大家都来了。</details> |
| 3 | Ngoài viết code, tôi còn phải họp. | <details><summary>🔑</summary>除了写代码以外，我还要开会。</details> |

**Điểm: ___/8**

## Bài Tập 5: Đọc Hiểu Tech (10 phút)

```
我们公司在开发一个新的手机应用。这个应用的
操作系统是Android和iOS。
我负责后端开发，用的编程语言是Java。
除了Java以外，有时候我还要写一点SQL。

上个星期系统出了一个错误。程序不能正常运行。
我查了两个小时，终于找到了问题。原来是
数据库的连接有问题。我把代码修好了，
然后重新部署到服务器上。

现在系统已经正常运行了。经理很满意。
```

| # | 问题 | Đáp án |
|---|------|--------|
| 1 | 公司在开发什么？ | <details><summary>🔑</summary>一个新的手机应用</details> |
| 2 | 作者负责什么？用什么语言？ | <details><summary>🔑</summary>后端开发，用Java</details> |
| 3 | 除了Java，他还写什么？ | <details><summary>🔑</summary>SQL</details> |
| 4 | 系统出了什么问题？ | <details><summary>🔑</summary>程序不能正常运行（数据库连接有问题）</details> |
| 5 | 他怎么解决的？ | <details><summary>🔑</summary>找到问题→修好代码→重新部署</details> |
