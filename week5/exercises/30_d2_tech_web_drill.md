## Bài Tập 2: Web Flow — Sắp Xếp Đúng Thứ Tự (10 phút)

Sắp xếp theo đúng luồng web request:

```
A. 浏览器在页面上显示数据
B. 用户在浏览器里输入网址
C. 服务器从数据库里读取数据
D. 浏览器向服务器发送请求
E. 服务器把数据返回给浏览器
F. 服务器收到请求
```

<details><summary>🔑 Đáp án</summary>B → D → F → C → E → A</details>

```
我们公司的网站有一百万用户。每天大概有十万人
登录。用户登录需要输入用户名和密码。

上个星期，有用户说登录页面显示错误。我查了
前端代码，发现是一个参数传错了。浏览器发送
请求的时候，密码参数没有加密。

我把这个bug修好了。现在登录功能已经正常了。
```
