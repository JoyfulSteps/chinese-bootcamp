## 📐 Phần 1: Ngữ Pháp — 除了...以外 (25 phút)

### Pattern 1: 除了 A 以外，也/还 B (ngoài A, còn B — THÊM)

```
除了 chúle = ngoài...ra
以外 yǐwài = bên ngoài
也/还 yě/hái = cũng/còn

除了Java以外，我还会写Python。
Chúle Java yǐwài, wǒ hái huì xiě Python.
Ngoài Java ra, tôi CÒN biết viết Python.

除了写代码以外，我还要开会。
Ngoài viết code ra, tôi CÒN phải họp.

除了中文以外，他也会说英文和越南语。
Ngoài TQ ra, anh ấy CŨNG biết nói Anh và VN.
```

### Pattern 2: 除了 A 以外，都 B (ngoài A, ai/gì cũng — TRỪ)

```
除了他以外，大家都来了。
Chúle tā yǐwài, dàjiā dōu lái le.
Ngoài anh ấy ra, MỌI NGƯỜI đều đến. (= chỉ anh ấy không đến)

除了星期天以外，我每天都上班。
Ngoài Chủ nhật ra, tôi NGÀY NÀO cũng đi làm.

除了这个bug以外，代码都没有问题。
Ngoài bug này ra, code đều không có vấn đề.
```

### Phân biệt: 也/还 vs 都

```
也/还 = THÊM (addition): A cũng, B cũng
  除了A以外，还有B。= Ngoài A, CÒN CÓ B.

都 = TRỪ (exclusion): trừ A ra, tất cả đều...
  除了A以外，都B。= Trừ A ra, TẤT CẢ đều B.
```

> 💡 **Dev tip:** `除了...也/还` = `array.concat()` — thêm phần tử. `除了...都` = `array.filter(x => x !== A)` — loại trừ phần tử.

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 系 xì (hệ/liên) — 7 nét
```
系统 = hệ thống, 关系 = quan hệ, 联系 = liên lạc
Tập viết: 系 系 系 系 系 系 系 系 系 系
```

#### 统 tǒng (thống) — 9 nét
```
纟(tơ) + 充(đầy) → "tơ kết nối" = THỐNG
系统 = hệ thống, 统一 = thống nhất
Tập viết: 统 统 统 统 统 统 统 统 统 统
```

#### 应 yìng (ứng) — 7 nét
```
应用 = ứng dụng, 应该 = nên (đã học!)
Tập viết: 应 应 应 应 应 应 应 应 应 应
```

#### 编 biān (biên/soạn) — 12 nét
```
纟(tơ) + 扁(dẹp) → "đan dệt tơ = biên soạn"
编程 = lập trình, 编辑 = biên tập/edit
Tập viết: 编 编 编 编 编 编 编 编 编 编
```

#### 程 chéng (trình/quá trình) — 12 nét
```
禾(lúa) + 呈(trình bày) → "quá trình trưởng thành"
编程 = lập trình, 程序 = chương trình, 程序员 = dev
Tập viết: 程 程 程 程 程 程 程 程 程 程
```

#### 装 zhuāng (trang/cài) — 12 nét
```
衣(áo) + 壮(mạnh) → "mặc áo = TRANG bị"
安装 = cài đặt, 装修 = trang trí
Tập viết: 装 装 装 装 装 装 装 装 装 装
```

#### 载 zài (tải/chở) — 10 nét
```
车(xe) + 戈(giáo) → "xe chở = TẢI"
下载 = tải xuống, 上载 = upload
Tập viết: 载 载 载 载 载 载 载 载 载 载
```

#### 误 wù (nhầm/sai) — 9 nét
```
讠(lời) + 吴 → "lời sai = SAI LẦM"
错误 = lỗi/bug, 误解 = hiểu nhầm
Tập viết: 误 误 误 误 误 误 误 误 误 误
```
