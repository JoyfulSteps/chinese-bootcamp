# 🇨🇳 Chinese Bootcamp — Tổng Quan Khóa Học

> **Người học:** Lập trình viên, tiếng Anh đọc hiểu tốt, chưa biết tiếng Trung
> **Thời gian:** 8 tuần (2 tháng) | 4–5 giờ/ngày | Tổng: ~250 giờ
> **Bắt đầu:** 2026-03-10 (Thứ 2)  

---

## 🧠 Triết Lý Khóa Học

### Vấn đề gốc rễ
Tiếng Trung **không phải** ngôn ngữ alphabet — không thể "đánh vần" như tiếng Anh/Việt.
→ Cần approach hoàn toàn khác: **Nghe trước, Đọc sau, Viết cuối.**

### Giải pháp: Multi-Skill Parallel Approach

```
Không phải: Học xong phát âm → rồi mới học từ vựng → rồi mới ngữ pháp
Mà là:      Mỗi ngày song song TẤT CẢ kỹ năng, tăng dần độ khó

Day 1: Pinyin cơ bản + 10 từ + 2 ngữ pháp + nghe 10 phút
Day 2: Pinyin tiếp + 10 từ mới + ôn 10 từ cũ + 2 ngữ pháp + nghe 15 phút
...
Day 56: HSK 3 vocab + đọc tech blog + nghe podcast + nói 5 phút
```

### 4 Nguyên Tắc Cốt Lõi

| # | Nguyên Tắc | Áp dụng cho tiếng Trung |
|---|-----------|------------------------|
| 1 | **Thực hành > Lý thuyết** | 70% luyện (nghe, nói, đọc, viết), 30% học lý thuyết |
| 2 | **Chất lượng > Số lượng** | 10 từ nhớ kỹ > 50 từ quên ngay. Phát âm đúng 1 thanh > qua loa 4 thanh |
| 3 | **Tích lũy dần (Spaced Repetition)** | Anki hàng ngày, review cũ trước khi học mới |
| 4 | **Tự đánh giá** | Cuối ngày: checklist. Cuối tuần: assessment report |

### Phương pháp áp dụng từ English Bootcamp

| Phương pháp gốc | Cách áp dụng cho tiếng Trung |
|-----------------|------------------------------|
| L1 First (học tư duy bằng tiếng mẹ đẻ trước) | Hiểu ngữ pháp TQ qua giải thích tiếng Việt, dùng tiếng Anh cho tài liệu nâng cao |
| 5 lớp mở rộng câu | Áp dụng từ tuần 3: mở rộng câu tiếng Trung (thêm thời gian, địa điểm, lý do...) |
| 8 cách biến đổi cấu trúc | Tuần 4+: biến đổi câu dùng 把, 被, 是...的 |
| PEEL Structure | Tuần 6+: viết đoạn văn tech tiếng Trung theo PEEL |
| Feedback flow (sửa từng câu) | Mỗi bài tập viết → AI review → file feedback |
| Anki Integration | Từ vựng + Hán tự + Ngữ pháp patterns, review hàng ngày |
| Progress Tracker | Cuối mỗi ngày cập nhật, cuối tuần assessment |

## 🏗️ Kiến Trúc 8 Tuần

```
PHASE 1: FOUNDATION (Tuần 1–2)        ═══════════════════
  Tuần 1: Pinyin + Thanh điệu + 80 từ HSK 1 + Nét Hán tự
  Tuần 2: Pinyin nâng cao + 70 từ HSK 1 còn lại + Ngữ pháp cơ bản
  → Checkpoint: Pass HSK 1 mock test ≥ 60%

PHASE 2: CORE BUILD (Tuần 3–4)        ═══════════════════
  Tuần 3: HSK 2 vocab (75 từ) + Ngữ pháp patterns + Đọc đoạn ngắn
  Tuần 4: HSK 2 vocab (75 từ) + Câu phức + Nghe hội thoại
  → Checkpoint: Pass HSK 2 mock test ≥ 60%

PHASE 3: EXPANSION (Tuần 5–6)         ═══════════════════
  Tuần 5: HSK 3 vocab (100 từ) + Tech Chinese cơ bản + Đọc blog
  Tuần 6: HSK 3 vocab (100 từ) + Viết đoạn + Nghe podcast
  → Checkpoint: Đọc hiểu bài tech 60%+

PHASE 4: APPLIED (Tuần 7–8)           ═══════════════════
  Tuần 7: HSK 3 hoàn thành + Nói tự giới thiệu + Đọc Juejin/CSDN
  Tuần 8: Tổng ôn + Mock interview + Final assessment
  → Checkpoint: Tự giới thiệu 2 phút bằng tiếng Trung
```

## ⏰ Cấu Trúc Ngày Học (4.5 giờ)

### 3 Sessions/ngày

```
┌────────────────────────────────────────────────────┐
│ SESSION 1: BUỔI SÁNG (1.5h)                       │
│   ├── Block A: Ôn tập Anki cũ (30 phút)           │
│   ├── Block B: Học từ vựng mới (30 phút)           │
│   └── Block C: Ngữ pháp/Kỹ năng mới (30 phút)    │
│                                                     │
│ SESSION 2: BUỔI CHIỀU (1.5h)                       │
│   ├── Block D: Luyện nghe + Shadowing (30 phút)    │
│   ├── Block E: Đọc hiểu (30 phút)                  │
│   └── Block F: Bài tập thực hành (30 phút)         │
│                                                     │
│ SESSION 3: BUỔI TỐI (1.5h)                         │
│   ├── Block G: Hán tự / Viết (30 phút)             │
│   ├── Block H: Luyện nói (30 phút)                 │
│   └── Block I: Review ngày + Anki tối (30 phút)    │
└────────────────────────────────────────────────────┘
```
