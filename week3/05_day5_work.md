## 📐 Phần 1: Ngữ Pháp — 因为...所以... (Vì...Nên...) (30 phút)

### Pattern: 因为 A，所以 B

```
因为 yīnwèi = vì (nguyên nhân)
所以 suǒyǐ  = nên/cho nên (kết quả)

因为下雨了，所以我没去上班。
Yīnwèi xià yǔ le, suǒyǐ wǒ méi qù shàngbān.
VÌ mưa, NÊN tôi không đi làm.

因为太贵了，所以我没买。
Yīnwèi tài guì le, suǒyǐ wǒ méi mǎi.
VÌ quá đắt, NÊN tôi không mua.

因为他很忙，所以没来。
Yīnwèi tā hěn máng, suǒyǐ méi lái.
VÌ anh ấy bận, NÊN không đến.

因为我喜欢中文，所以我每天学。
Yīnwèi wǒ xǐhuan Zhōngwén, suǒyǐ wǒ měi tiān xué.
VÌ tôi thích TQ, NÊN tôi học mỗi ngày.
```

### ⚠️ Có thể bỏ 1 phần (nói tự nhiên hơn)

```
Nói đầy đủ:  因为下雨了，所以我没去。
Bỏ 因为:     下雨了，所以我没去。     (vẫn OK!)
Bỏ 所以:     因为下雨了，我没去。     (vẫn OK!)
Bỏ cả 2:    ❌ (phải giữ ít nhất 1 cái)

→ Trong nói hàng ngày, người TQ hay bỏ 因为, chỉ nói 所以:
  "太贵了，所以没买。"  (Đắt quá, nên không mua.)
```

### Hỏi lý do: 为什么 wèishénme = Tại sao?

```
你为什么学中文？       Nǐ wèishénme xué Zhōngwén?
Bạn TẠI SAO học TQ?

→ 因为我想去中国工作。 Yīnwèi wǒ xiǎng qù Zhōngguó gōngzuò.
   Vì tôi muốn đi TQ làm việc.

你为什么没来？         Nǐ wèishénme méi lái?
Tại sao bạn không đến?

→ 因为我生病了。       Yīnwèi wǒ shēng bìng le.
   Vì tôi bị bệnh.
```

> 💡 **Dev tip:** `因为...所以...` = `if (cause) { result }`. `为什么` = debugger asking "WHY did this happen?"

## 📚 Phần 2: Nghề Nghiệp (20 phút)

### Các nghề phổ biến

|--------|--------|-------|---------|
| **程序员** | chéngxùyuán | lập trình viên | ← ĐÂY LÀ BẠN! |
| **老师** | lǎoshī | giáo viên | Đã học W1 |
| **医生** | yīshēng | bác sĩ | Đã học W2 |
| **工程师** | gōngchéngshī | kỹ sư | 工程 = công trình |
| **经理** | jīnglǐ | giám đốc/quản lý | 总经理 = tổng GĐ |

| **服务员** | fúwùyuán | nhân viên phục vụ | Đã học D18 |
| **警察** | jǐngchá | cảnh sát | |

| **护士** | hùshi | y tá | |

### Nói về công việc

```
你做什么工作？       Nǐ zuò shénme gōngzuò?    Bạn làm nghề gì?
我是程序员。         Wǒ shì chéngxùyuán.       Tôi là lập trình viên.
我在一家公司工作。   Wǒ zài yì jiā gōngsī gōngzuò.  Tôi làm ở 1 công ty.

你的工作怎么样？     Nǐ de gōngzuò zěnmeyàng?  Công việc thế nào?
很忙，但是很有意思。 Hěn máng, dànshì hěn yǒu yìsi.  Bận nhưng thú vị.

你为什么当程序员？   Nǐ wèishénme dāng chéngxùyuán?  Tại sao làm dev?
因为我喜欢写代码。   Yīnwèi wǒ xǐhuan xiě dàimǎ.    Vì tôi thích code.
```

### Từ vựng công sở

```

项目     xiàngmù      dự án

工资     gōngzī       lương
```

## 📖 Phần 3: 13 Từ Vựng Mới (25 phút)

> Tổng tích lũy sau Day 19: **213 từ**

| # | Hán tự | Pinyin | Nghĩa | Câu ví dụ |
|---|--------|--------|-------|-----------|
| 201 | **因为** | yīnwèi | vì | 因为下雨了... |
| 202 | **所以** | suǒyǐ | nên/cho nên | ...所以我没去。 |
| 203 | **为什么** | wèishénme | tại sao | 你为什么学中文？ |
| 204 | **喜欢** | xǐhuan | thích | 我喜欢中文。 |
| 205 | **同事** | tóngshì | đồng nghiệp | 我跟同事一起吃饭。 |
| 206 | **开会** | kāi huì | họp | 下午要开会。(Chiều phải họp.) |
| 207 | **加班** | jiā bān | tăng ca | 今天要加班。(Hôm nay phải OT.) |
| 208 | **每** | měi | mỗi | 每天 (mỗi ngày), 每个人 (mỗi người) |
| 209 | **对** | duì | đúng/đối với | 对不起 (xin lỗi), 你说得对 (bạn nói đúng) |
| 210 | **错** | cuò | sai | 你说错了。(Bạn nói sai rồi.) |
| 211 | **觉得** | juéde | cảm thấy/nghĩ | 我觉得很好。(Tôi thấy rất tốt.) |

### Cụm từ thực hành

```
因为我喜欢写代码，所以我当了程序员。
Vì tôi thích code, nên tôi làm lập trình viên.

你为什么加班？因为项目很忙。
Tại sao OT? Vì dự án bận.

我觉得这个工作很好。
Tôi thấy công việc này rất tốt.

每天我跟同事一起吃午饭。
Mỗi ngày tôi cùng đồng nghiệp ăn trưa.

他可能今天不来开会。
Anh ấy có lẽ hôm nay không đến họp.

你说得对！/ 你说错了！
Bạn nói đúng! / Bạn nói sai rồi!
```

## 🗣️ Phần 5: Hội Thoại — Nói Về Công Việc (15 phút)

```
A: 你好！你做什么工作？
   Nǐ hǎo! Nǐ zuò shénme gōngzuò?

B: 我是程序员，在一家公司工作。你呢？
   Wǒ shì chéngxùyuán, zài yì jiā gōngsī gōngzuò. Nǐ ne?

A: 我是老师。你的工作怎么样？
   Wǒ shì lǎoshī. Nǐ de gōngzuò zěnmeyàng?

B: 很忙，每天要开会、写代码。有时候要加班。
   Hěn máng, měi tiān yào kāi huì, xiě dàimǎ. Yǒu shíhou yào jiā bān.

A: 你喜欢你的工作吗？
   Nǐ xǐhuan nǐ de gōngzuò ma?

B: 喜欢！因为我喜欢写代码。你呢？
   Xǐhuan! Yīnwèi wǒ xǐhuan xiě dàimǎ. Nǐ ne?

A: 我也很喜欢当老师。每天跟学生在一起，很开心。
   Wǒ yě hěn xǐhuan dāng lǎoshī. Měi tiān gēn xuésheng zài yìqǐ, hěn kāixīn.

B: 太好了！做自己喜欢的工作最重要。
```

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 因 yīn (nhân/vì) — 6 nét
```
囗(bao) + 大(lớn) → "nguyên nhân lớn" = VÌ/NHÂN
因为 = vì
Tập viết: 因 因 因 因 因 因 因 因 因 因
```

#### 为 wèi/wéi (vì/cho) — 4 nét
```
因为 = vì, 为什么 = tại sao
Tập viết: 为 为 为 为 为 为 为 为 为 为
```

#### 所 suǒ (chỗ/thì) — 8 nét
```
所以 = nên/cho nên
Tập viết: 所 所 所 所 所 所 所 所 所 所
```

#### 喜 xǐ (vui/thích) — 12 nét
```
壴(trống) + 口(miệng) → "miệng cười" = VUI/THÍCH
喜欢 = thích
Tập viết: 喜 喜 喜 喜 喜 喜 喜 喜 喜 喜
```

#### 欢 huān (vui) — 6 nét
```
又(tay) + 欠(thiếu) → giản thể của 歡
喜欢 = thích, 欢迎 = chào mừng
Tập viết: 欢 欢 欢 欢 欢 欢 欢 欢 欢 欢
```

#### 同 tóng (cùng) — 6 nét
```
同事 = đồng nghiệp, 同学 = bạn cùng lớp
不同 = không giống nhau
Tập viết: 同 同 同 同 同 同 同 同 同 同
```

#### 错 cuò (sai) — 13 nét
```
钅(kim loại) + 昔(xưa) → "kim loại cũ = SAI lầm"
你说错了 = bạn nói sai rồi
Tập viết: 错 错 错 错 错 错 错 错 错 错
```

#### 每 měi (mỗi) — 7 nét
```
每天 = mỗi ngày, 每个人 = mỗi người
Tập viết: 每 每 每 每 每 每 每 每 每 每
```
