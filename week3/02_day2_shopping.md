## 📐 Phần 1: Ngữ Pháp — 比 So Sánh (30 phút)

### Pattern cơ bản: A 比 B + Adj

```
比 bǐ = "hơn" (so với)

A 比 B + Adj = A [adj] hơn B

  他比我高。       Tā bǐ wǒ gāo.         Anh ấy CAO hơn tôi.
  今天比昨天热。   Jīntiān bǐ zuótiān rè. Hôm nay NÓNG hơn hôm qua.
  飞机比火车快。   Fēijī bǐ huǒchē kuài.  Máy bay NHANH hơn tàu hỏa.
  这个比那个贵。   Zhège bǐ nàge guì.     Cái này ĐẮT hơn cái kia.
  中文比英文难。   Zhōngwén bǐ Yīngwén nán. TQ KHÓ hơn tiếng Anh.
```

### ⚠️ QUY TẮC QUAN TRỌNG: 比 KHÔNG dùng với 很!

```
❌ 他比我很高。    (SAI! Không thêm 很 vào 比)
✅ 他比我高。      (ĐÚNG)

❌ 今天比昨天很热。  (SAI!)
✅ 今天比昨天热。    (ĐÚNG)
```

### Thêm mức độ: 一点 (hơn chút) / 多了 (hơn nhiều)

```
A 比 B + Adj + 一点 = hơn MỘT CHÚT
  这个比那个贵一点。    Cái này đắt hơn cái kia MỘT CHÚT.
  他比我大一点。        Anh ấy lớn hơn tôi một chút.

A 比 B + Adj + 多了 = hơn RẤT NHIỀU
  飞机比火车快多了！    Máy bay nhanh hơn tàu hỏa RẤT NHIỀU!
  这个比那个贵多了！    Cái này đắt hơn cái kia nhiều!

A 比 B + Adj + 数字 = hơn CỤ THỂ
  他比我大三岁。        Anh ấy lớn hơn tôi 3 tuổi.
  这个比那个贵五块。    Cái này đắt hơn cái kia 5 đồng.
```

### Phủ định: 没有...那么... (không bằng)

```
⚠️ KHÔNG nói: A 不比 B (tuy đúng ngữ pháp nhưng ít dùng)
✅ Thay bằng: A 没有 B + 那么 + Adj

  我没有他那么高。       Wǒ méi yǒu tā nàme gāo.
  Tôi không cao BẰNG anh ấy.

  今天没有昨天那么热。   Jīntiān méi yǒu zuótiān nàme rè.
  Hôm nay không nóng BẰNG hôm qua.
```

> 💡 **Dev tip:** `比` giống `>` operator: `A 比 B 高` = `A.height > B.height`. Phủ định `没有那么` = `<=`.

## 📚 Phần 2: Mua Sắm & Giá Cả (25 phút)

### Cách hỏi giá

```
多少钱？           Duōshao qián?           Bao nhiêu tiền?
这个多少钱？       Zhège duōshao qián?     Cái này bao nhiêu?
那个呢？           Nàge ne?                Cái kia thì sao?
太贵了！           Tài guì le!             Quá đắt!
便宜一点吧？       Piányi yìdiǎn ba?       Rẻ hơn chút đi?
可以便宜一点吗？   Kěyǐ piányi yìdiǎn ma?  Có thể rẻ hơn không?
```

### Đơn vị tiền TQ

```
元 yuán (formal) / 块 kuài (nói thường) = Nhân dân tệ (RMB/¥)
角 jiǎo (formal) / 毛 máo (nói thường) = 1/10 của 元
分 fēn = 1/100 của 元

15 块 3 毛 = ¥15.30 = mười五块三(毛)
一百块 = ¥100

Nói giá:
  十五块      shíwǔ kuài        ¥15
  两百块      liǎng bǎi kuài    ¥200
  三十五块五  sānshíwǔ kuài wǔ  ¥35.50
```

### Màu sắc cơ bản

```
红色 hóngsè   đỏ        蓝色 lánsè    xanh dương
黄色 huángsè  vàng       绿色 lǜsè     xanh lá
白色 báisè    trắng      黑色 hēisè    đen

Dùng: 红色的 + danh từ
  红色的手机     ĐT đỏ
  黑色的电脑     MT đen
  你要什么颜色？  Bạn muốn màu gì?
```

## 🗣️ Phần 3: Hội Thoại — Mua Hàng (20 phút)

### Hội thoại: Ở cửa hàng

```
顾客(A): 你好！这个手机多少钱？
         Nǐ hǎo! Zhège shǒujī duōshao qián?

店员(B): 这个两千块。
         Zhège liǎng qiān kuài.

A: 太贵了！那个呢？
   Tài guì le! Nàge ne?

B: 那个一千五百块，比这个便宜一点。
   Nàge yìqiān wǔbǎi kuài, bǐ zhège piányi yìdiǎn.

A: 那个是什么颜色的？
   Nàge shì shénme yánsè de?

B: 有黑色的和白色的。
   Yǒu hēisè de hé báisè de.

A: 我要黑色的。可以便宜一点吗？
   Wǒ yào hēisè de. Kěyǐ piányi yìdiǎn ma?

B: 好吧，一千四百块。
   Hǎo ba, yìqiān sìbǎi kuài.

A: 好的，我买了！谢谢！
   Hǎo de, wǒ mǎi le! Xièxie!
```

> 🎤 **Shadowing:** Nghe hội thoại (gõ GT) → đọc theo 5 lần. Thử đóng vai 顾客(khách) mua hàng!

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 比 bǐ (so sánh) — 4 nét
```
Hai người đứng cạnh nhau → SO SÁNH
Tập viết: 比 比 比 比 比 比 比 比 比 比
```

#### 红 hóng (đỏ) — 6 nét
```
纟(sợi tơ) + 工(công) → tơ lụa đỏ = ĐỎ
红色 = màu đỏ
Tập viết: 红 红 红 红 红 红 红 红 红 红
```

#### 白 bái (trắng) — 5 nét
```
Hình mặt trời trắng lúc bình minh
白色 = trắng, 白天 = ban ngày
Tập viết: 白 白 白 白 白 白 白 白 白 白
```

#### 黑 hēi (đen) — 12 nét
```
Phần trên: 里 biến thể + bốn chấm lửa dưới → "cháy = ĐEN"
黑色 = đen
Tập viết: 黑 黑 黑 黑 黑 黑 黑 黑 黑 黑
```

#### 穿 chuān (mặc) — 9 nét
```
穴(hang) + 牙(răng) → "chui qua hang" = MẶC (xuyên qua)
穿衣服 = mặc quần áo
Tập viết: 穿 穿 穿 穿 穿 穿 穿 穿 穿 穿
```

#### 给 gěi (cho/đưa) — 9 nét
```
纟(tơ) + 合(hợp) → "tơ hợp lại đưa cho" = CHO
给你 = đưa bạn, 给我 = đưa tôi
Tập viết: 给 给 给 给 给 给 给 给 给 给
```

#### 找 zhǎo (tìm/thối) — 7 nét
```
扌(tay) + 戈(giáo) → "tay cầm giáo tìm" = TÌM
找人 = tìm người, 找钱 = thối tiền
Tập viết: 找 找 找 找 找 找 找 找 找 找
```

#### 已 yǐ (đã) — 3 nét
```
Chỉ 3 nét! 已经 = đã (rồi)
Tập viết: 已 已 已 已 已 已 已 已 已 已
```
