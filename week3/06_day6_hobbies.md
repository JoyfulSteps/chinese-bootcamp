## 📐 Phần 1: Ngữ Pháp — 虽然...但是... (Tuy...Nhưng...) (25 phút)

### Pattern: 虽然 A，但是 B

```
虽然 suīrán = tuy/mặc dù
但是 dànshì = nhưng mà (đã học D17)

虽然很难，但是很有意思。
Suīrán hěn nán, dànshì hěn yǒu yìsi.
TUY rất khó, NHƯNG rất thú vị.

虽然加班很累，但是工资很高。
Suīrán jiā bān hěn lèi, dànshì gōngzī hěn gāo.
TUY OT mệt, NHƯNG lương cao.

虽然今天下雨了，但是我还是去跑步了。
Suīrán jīntiān xià yǔ le, dànshì wǒ háishi qù pǎo bù le.
TUY hôm nay mưa, NHƯNG tôi vẫn đi chạy bộ.

虽然他很忙，但是每天都学中文。
Suīrán tā hěn máng, dànshì měi tiān dōu xué Zhōngwén.
TUY anh ấy bận, NHƯNG mỗi ngày đều học TQ.
```

### ⚠️ 虽然 vs 因为 — khác nhau!

```
因为...所以... = NGUYÊN NHÂN → KẾT QUẢ
  因为下雨了，所以我没去。    (Vì mưa → nên không đi.)

虽然...但是... = NHƯỢNG BỘ → NGƯỢC LẠI
  虽然下雨了，但是我还是去了。(Tuy mưa → nhưng vẫn đi.)
```

> 💡 **Dev tip:** 
> `因为所以` = `if (rain) { don't_go(); }` — logic thuận
> `虽然但是` = `try { go(); } catch (rain) { still_go(); }` — exception handling!

### Kết hợp 因为 + 虽然

```
因为我喜欢中文，所以每天学。虽然很难，但是很有意思。
Vì tôi thích TQ, nên mỗi ngày học. Tuy khó, nhưng thú vị.
```

### 打 vs 踢 vs 玩

```
打 dǎ = đánh (dùng tay hoặc vợt):
  打篮球 (bóng rổ), 打网球 (tennis), 打乒乓球 (bóng bàn)

踢 tī = đá (dùng chân):
  踢足球 (bóng đá)

玩 wán = chơi (chung):
  玩游戏 (chơi game), 玩电脑 (chơi máy tính)
```

### Mẫu câu nói sở thích

```
你有什么爱好？         Nǐ yǒu shénme àihào?     Bạn có sở thích gì?
你喜欢做什么？         Nǐ xǐhuan zuò shénme?    Bạn thích làm gì?

我喜欢看书和听音乐。   Wǒ xǐhuan kàn shū hé tīng yīnyuè.
Tôi thích đọc sách và nghe nhạc.

我最喜欢跑步。         Wǒ zuì xǐhuan pǎo bù.    Tôi thích chạy bộ nhất.

我对中国文化很感兴趣。 Wǒ duì Zhōngguó wénhuà hěn gǎn xìngqù.
Tôi rất hứng thú với văn hóa TQ.

我不太喜欢运动。       Wǒ bú tài xǐhuan yùndòng.  Tôi không thích thể thao lắm.
```

### 对...感兴趣 = hứng thú với...

```
我对编程很感兴趣。     Tôi rất hứng thú với lập trình.
他对中文很感兴趣。     Anh ấy rất hứng thú với TQ.
你对什么感兴趣？       Bạn hứng thú với gì?
```

   Nǐ yǒu shénme àihào?

   Wǒ xǐhuan dǎ lánqiú. Wǒ měi ge zhōumò dōu dǎ.

   Nǐ dǎ de hǎo ma?

   Hái xíng ba. Suīrán bú tài hǎo, dànshì hěn kāixīn.

   Wǒ duì yóu yǒng yě hěn gǎn xìngqù, dànshì méi yǒu shíjiān qù.

B: 有时间的话，我们一起去游泳吧！
   Yǒu shíjiān de huà, wǒmen yìqǐ qù yóu yǒng ba!

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 虽 suī (tuy) — 9 nét
```
虽然 = tuy/mặc dù
Tập viết: 虽 虽 虽 虽 虽 虽 虽 虽 虽 虽
```

#### 然 rán (nhiên) — 12 nét
```
虽然 = tuy, 然后 = rồi/sau đó, 当然 = tất nhiên
Tập viết: 然 然 然 然 然 然 然 然 然 然
```

#### 运 yùn (vận) — 7 nét
```
辶(đi) + 云(mây) → "mây bay đi = VẬN chuyển"
运动 = thể thao/vận động
Tập viết: 运 运 运 运 运 运 运 运 运 运
```

#### 动 dòng (động) — 6 nét
```
云 + 力(sức) → "mây với sức = ĐỘNG"
运动 = thể thao, 动物 = động vật
Tập viết: 动 动 动 动 动 动 动 动 动 动
```

#### 歌 gē (bài hát) — 14 nét
```
哥(anh) + 欠(thiếu) → "anh thiếu bài hát" → BÀI HÁT
唱歌 = hát, 歌曲 = bài hát
Tập viết: 歌 歌 歌 歌 歌 歌 歌 歌 歌 歌
```

#### 跑 pǎo (chạy) — 12 nét
```
足(chân) + 包(gói) → "chân bước nhanh" = CHẠY
跑步 = chạy bộ
Tập viết: 跑 跑 跑 跑 跑 跑 跑 跑 跑 跑
```

#### 玩 wán (chơi) — 8 nét
```
王(vua) + 元(gốc) → "vua giải trí" = CHƠI
玩游戏 = chơi game
Tập viết: 玩 玩 玩 玩 玩 玩 玩 玩 玩 玩
```

#### 球 qiú (bóng) — 11 nét
```
王(ngọc) + 求(cầu) → "viên ngọc tròn" = BÓNG
篮球 = bóng rổ, 足球 = bóng đá
Tập viết: 球 球 球 球 球 球 球 球 球 球
```
