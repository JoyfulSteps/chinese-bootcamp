## 📐 Phần 1: Ngữ Pháp — 给 (Cho / Đưa cho) (25 phút)

### Pattern 1: 给 + Ai + Cái gì (Cho ai cái gì)

```
给 gěi = cho / đưa cho

给你。           Gěi nǐ.              Đưa cho bạn. / Cho bạn.
给我一杯水。     Gěi wǒ yì bēi shuǐ.  Cho tôi 1 cốc nước.
给他钱。         Gěi tā qián.          Đưa tiền cho anh ấy.
妈妈给我一本书。 Māma gěi wǒ yì běn shū. Mẹ cho tôi 1 cuốn sách.
```

### Pattern 2: Ai 给 Ai + V (Ai làm gì cho ai)

```
他给我买了一个手机。  Tā gěi wǒ mǎi le yí ge shǒujī.
Anh ấy mua cho tôi 1 cái ĐT.

妈妈给我做饭。      Māma gěi wǒ zuò fàn.
Mẹ nấu cơm cho tôi.

我给你写一封信。    Wǒ gěi nǐ xiě yì fēng xìn.
Tôi viết cho bạn 1 lá thư.
```

### Trong nhà hàng: 给我... = Cho tôi...

```
请给我一杯茶。      Qǐng gěi wǒ yì bēi chá.     Cho tôi 1 cốc trà.
请给我菜单。        Qǐng gěi wǒ càidān.          Cho tôi menu.
给我买单。          Gěi wǒ mǎidān.               Cho tôi tính tiền.
```

## 📐 Phần 2: Ngữ Pháp — 跟 (Với / Và) (20 phút)

### Pattern: A 跟 B + V (A làm V với B)

```
跟 gēn = với / cùng

我跟朋友去吃饭。      Wǒ gēn péngyou qù chī fàn.
Tôi VỚI bạn đi ăn cơm.

他跟我说话。          Tā gēn wǒ shuō huà.
Anh ấy nói chuyện VỚI tôi.

我想跟你一起去。      Wǒ xiǎng gēn nǐ yìqǐ qù.
Tôi muốn đi CÙNG bạn.

你跟谁去？            Nǐ gēn shéi qù?
Bạn đi VỚI ai?
```

### 跟 vs 和 — Cùng nghĩa "và/với"?

```
和 hé = VÀ (connecting nouns: A và B)
  我和他      wǒ hé tā        tôi VÀ anh ấy
  爸爸和妈妈  bàba hé māma     bố VÀ mẹ

跟 gēn = VỚI (doing something together)
  我跟他去    wǒ gēn tā qù    tôi đi VỚI anh ấy
  跟我说      gēn wǒ shuō     nói VỚI tôi

→ Nhiều trường hợp dùng thay nhau được:
  我和他去 = 我跟他去 (cả 2 đều OK)
```

> 💡 **Dev tip:** `和` ≈ `AND` operator (parallel). `跟` ≈ `WITH` modifier (together).

## 📚 Phần 3: Nhà Hàng & Đồ Ăn (20 phút)

### Từ vựng nhà hàng

```
服务员   fúwùyuán    nhân viên phục vụ (waiter)
菜单     càidān      menu
菜       cài         món ăn / rau
点菜     diǎn cài    gọi món
买单     mǎidān      tính tiền / check
```

### Đồ ăn phổ biến

```
米饭     mǐfàn       cơm
面条     miàntiáo     mì
鸡蛋     jīdàn        trứng gà
鸡肉     jī ròu       thịt gà
牛肉     niú ròu      thịt bò
鱼       yú           cá
蔬菜     shūcài       rau
水果     shuǐguǒ      hoa quả
```

### Câu hữu ích trong nhà hàng

```
服务员！        Fúwùyuán!            Phục vụ ơi!
请给我菜单。    Qǐng gěi wǒ càidān.  Cho tôi menu.
我要点菜。      Wǒ yào diǎn cài.     Tôi muốn gọi món.
这个是什么？    Zhège shì shénme?     Cái này là gì?
不要太辣。      Bú yào tài là.        Đừng quá cay.
好吃吗？        Hǎo chī ma?           Ngon không?
很好吃！        Hěn hǎo chī!          Rất ngon!
买单！          Mǎidān!               Tính tiền!
一共多少钱？    Yígòng duōshao qián?  Tổng bao nhiêu tiền?
```

## 🗣️ Phần 4: Hội Thoại — Đi Ăn Nhà Hàng (20 phút)

```
顾客(A+B): 你好！两个人。
           Nǐ hǎo! Liǎng ge rén.

服务员(C): 好的，这边请。请看菜单。
           Hǎo de, zhèbiān qǐng. Qǐng kàn càidān.

A: 我想吃鸡肉，你呢？
   Wǒ xiǎng chī jī ròu, nǐ ne?

B: 我要牛肉面条。再来一个蔬菜。
   Wǒ yào niú ròu miàntiáo. Zài lái yí ge shūcài.

A: 好的。服务员，我们要一个鸡肉、一个
   牛肉面条、一个蔬菜，两杯茶。
   Hǎo de. Fúwùyuán, wǒmen yào yí ge jī ròu,
   yí ge niú ròu miàntiáo, yí ge shūcài, liǎng bēi chá.

C: 好的，请等一下。

    ——— (上菜了 shàng cài le — lên món!) ———

C: 你们的菜来了。请慢用。
   Nǐmen de cài lái le. Qǐng màn yòng.

A: 这个鸡肉很好吃！
   Zhège jī ròu hěn hǎo chī!

B: 我的面条也很好吃！

A: 服务员，买单！一共多少钱？
C: 一共八十五块。
A: 给你一百块。
C: 找你十五块。谢谢！
A: 谢谢，再见！
```

> 🎤 **Shadowing:** Gõ vào GT → nghe → đọc theo. Đóng vai khách hàng 5 lần!

## ✏️ Phần 7: 8 Hán Tự Mới (20 phút)

#### 给 gěi (cho) — 9 nét
```
纟(tơ) + 合(hợp) → "tơ hợp đưa cho" = CHO
给你 = cho bạn, 给我 = cho tôi
Tập viết: 给 给 给 给 给 给 给 给 给 给
```

#### 菜 cài (món ăn/rau) — 11 nét
```
艹(cỏ) + 采(hái) → "hái cỏ/rau" = MÓN ĂN/RAU
点菜 = gọi món, 菜单 = menu
Tập viết: 菜 菜 菜 菜 菜 菜 菜 菜 菜 菜
```

#### 鱼 yú (cá) — 8 nét
```
Giản thể của 魚. Hình con cá
吃鱼 = ăn cá
Tập viết: 鱼 鱼 鱼 鱼 鱼 鱼 鱼 鱼 鱼 鱼
```

#### 肉 ròu (thịt) — 6 nét
```
Hình miếng thịt
鸡肉 = thịt gà, 牛肉 = thịt bò
Tập viết: 肉 肉 肉 肉 肉 肉 肉 肉 肉 肉
```

#### 等 děng (chờ/đợi) — 12 nét
```
竹(tre) + 寺(chùa) → "ngồi dưới tre chờ" = CHỜ
等一下 = chờ chút, 请等 = xin chờ
Tập viết: 等 等 等 等 等 等 等 等 等 等
```

#### 面 miàn (mặt/mì) — 9 nét
```
面条 = mì, 前面 = phía trước, 外面 = bên ngoài
Tập viết: 面 面 面 面 面 面 面 面 面 面
```

#### 起 qǐ (dậy/bắt đầu) — 10 nét
```
走(đi) + 己(mình) → "mình đứng DẬY đi"
起床 = dậy, 一起 = cùng nhau
Tập viết: 起 起 起 起 起 起 起 起 起 起
```

#### 共 gòng (cùng/chung) — 6 nét
```
一共 = tổng cộng, 公共 = công cộng
Tập viết: 共 共 共 共 共 共 共 共 共 共
```
