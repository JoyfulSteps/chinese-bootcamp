## 📚 Phần 1: Phương Tiện Giao Thông (20 phút)

### Các phương tiện chính

| Hán tự | Pinyin | Nghĩa | Cách đi (动词) |
|--------|--------|-------|---------------|
| **公共汽车** | gōnggòng qìchē | xe bus | 坐 zuò (ngồi) |
| **出租车** | chūzūchē | taxi | 坐 zuò |
| **地铁** | dìtiě | tàu điện ngầm/metro | 坐 zuò |
| **火车** | huǒchē | tàu hỏa | 坐 zuò |
| **飞机** | fēijī | máy bay | 坐 zuò |
| **自行车** | zìxíngchē | xe đạp | 骑 qí (cưỡi) |
| **汽车** | qìchē | ô tô | 开 kāi (lái) |

### 3 động từ đi phương tiện: 坐 / 骑 / 开

```
坐 zuò = NGỒI → phương tiện lớn (bus, taxi, tàu, máy bay)
  坐公共汽车    zuò gōnggòng qìchē   đi xe bus
  坐出租车      zuò chūzūchē          đi taxi
  坐地铁        zuò dìtiě             đi metro
  坐飞机        zuò fēijī             đi máy bay

骑 qí  = CƯỠI → phương tiện nhỏ 2 bánh
  骑自行车      qí zìxíngchē          đi xe đạp
  骑摩托车      qí mótuōchē           đi xe máy

开 kāi = LÁI → tự lái phương tiện
  开车          kāi chē               lái xe (ô tô)
  开出租车      kāi chūzūchē          lái taxi (là tài xế)

走路 zǒu lù = ĐI BỘ (không cần phương tiện)
```

> 💡 **Dev tip:** 
> `坐` = passenger mode (ngồi, người khác lái)
> `骑` = 2-wheel mode (cưỡi, 2 bánh)
> `开` = driver mode (lái, mình điều khiển)
> `走` = walk mode

## 📐 Phần 2: Ngữ Pháp — 从...到... (Từ...đến...) (25 phút)

### Pattern 1: 从 A 到 B (Từ A đến B — địa điểm)

```
从 + 地点A + 到 + 地点B
cóng          dào

从家到公司          Cóng jiā dào gōngsī         Từ nhà đến công ty
从越南到中国        Cóng Yuènán dào Zhōngguó    Từ VN đến TQ
从学校到商店        Cóng xuéxiào dào shāngdiàn  Từ trường đến CH
```

### Pattern 2: 从 A 到 B (Từ A đến B — thời gian)

```
从 + 时间A + 到 + 时间B

从星期一到星期五    Cóng xīngqī yī dào xīngqī wǔ   Từ thứ 2 đến thứ 6
从早上到晚上        Cóng zǎoshang dào wǎnshang      Từ sáng đến tối
从九点到六点        Cóng jiǔ diǎn dào liù diǎn      Từ 9h đến 6h
```

### Pattern 3: Kết hợp với phương tiện

```
从家到公司坐地铁。
Cóng jiā dào gōngsī zuò dìtiě.
Từ nhà đến công ty đi metro.

从越南到中国坐飞机。
Cóng Yuènán dào Zhōngguó zuò fēijī.
Từ VN đến TQ đi máy bay.

你从哪里来？
Nǐ cóng nǎli lái?
Bạn từ đâu đến? (= Bạn quê ở đâu?)
```

### Hỏi đường: 怎么去 (đi thế nào?)

```
你怎么去上班？       Nǐ zěnme qù shàngbān?      Bạn đi làm bằng gì?
→ 我坐地铁去上班。  Wǒ zuò dìtiě qù shàngbān.  Tôi đi metro đi làm.
→ 我骑自行车去。    Wǒ qí zìxíngchē qù.         Tôi đi xe đạp.
→ 我走路去。        Wǒ zǒu lù qù.               Tôi đi bộ.

从这里到学校怎么去？ Cóng zhèli dào xuéxiào zěnme qù?
Từ đây đến trường đi thế nào?
→ 坐公共汽车。      Zuò gōnggòng qìchē.         Đi xe bus.
```

## ✏️ Phần 6: 8 Hán Tự Mới (25 phút)

#### 车 chē (xe) — 4 nét
```
Giản thể từ 車 (hình xe cổ)
汽车 = ô tô, 火车 = tàu hỏa, 公共汽车 = bus
Tập viết: 车 车 车 车 车 车 车 车 车 车
```

#### 坐 zuò (ngồi) — 7 nét
```
两人(2 người) + 土(đất) → "2 người ngồi trên đất" = NGỒI
坐车 = đi xe (ngồi xe)
Tập viết: 坐 坐 坐 坐 坐 坐 坐 坐 坐 坐
```

#### 走 zǒu (đi) — 7 nét
```
土(đất) + 止(dừng) → "đất nâng chân bước" = ĐI
走路 = đi bộ
Tập viết: 走 走 走 走 走 走 走 走 走 走
```

#### 路 lù (đường) — 13 nét
```
足(chân) + 各(mỗi) → "mỗi bước chân = ĐƯỜNG"
Bộ thủ: 足 — liên quan CHÂN/ĐI
Tập viết: 路 路 路 路 路 路 路 路 路 路
```

#### 远 yuǎn (xa) — 7 nét
```
辶(đi) + 元 → "đi xa" = XA
Đối lập: 近 jìn (gần)
Tập viết: 远 远 远 远 远 远 远 远 远 远
```

#### 近 jìn (gần) — 7 nét
```
辶(đi) + 斤 → "đi gần" = GẦN
Đối lập: 远 yuǎn (xa)
Tập viết: 近 近 近 近 近 近 近 近 近 近
```

#### 左 zuǒ (trái) — 5 nét
```
工(công) + 一 → giống tay trái cầm dụng cụ
Đối lập: 右 yòu (phải)
Tập viết: 左 左 左 左 左 左 左 左 左 左
```

#### 右 yòu (phải) — 5 nét
```
口(miệng) + 一 → tay phải che miệng
Đối lập: 左 zuǒ (trái)
Tập viết: 右 右 右 右 右 右 右 右 右 右
```
