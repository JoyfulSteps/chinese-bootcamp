## Bài Tập 3: Gọi Món — Hoàn Thành Hội Thoại (15 phút)

```
顾客: ___！请___我菜单。       (phục vụ ơi! cho tôi menu)
服务员: 好的，请___。          (xin mời xem)
顾客: 我要一个___面条和一个___。 (mì bò, rau)
服务员: 好的。要___什么？       (uống gì?)
顾客: 两___茶。                 (2 cốc trà)
服务员: 好的，请___一下。       (xin chờ)

    ——— (上菜了) ———

顾客: 很___！                    (rất ngon!)
顾客: ___！一___多少钱？         (tính tiền! tổng bao nhiêu?)
服务员: ___四十五块。            (tổng 45 đồng)
顾客: ___你。                    (cảm ơn)
```

<details>
<summary>🔑 Đáp án</summary>

服务员, 给 / 看 / 牛肉, 蔬菜 / 喝 / 杯 / 等 / 好吃 / 买单, 共 / 一共 / 谢谢

</details>

## Bài Tập 4: Đọc Menu + Tính Tiền (10 phút)

```
你和朋友去"好吃饭店"。看菜单：
  米饭 ¥3  |  牛肉面条 ¥25  |  鸡蛋面条 ¥18
  鸡肉 ¥38  |  鱼 ¥45  |  蔬菜 ¥15
  茶 ¥5  |  咖啡 ¥15  |  水 ¥2
```

Gọi:

| Bạn gọi | Bạn ấy gọi |
|---------|-----------|
| 鸡蛋面条 + 蔬菜 + 茶 | 鸡肉 + 米饭 + 咖啡 |

1. Bạn gọi bao nhiêu tiền? ___块
2. Bạn ấy gọi bao nhiêu tiền? ___块
3. 一共___块

<details>
<summary>🔑 Đáp án</summary>

Bạn: 18 + 15 + 5 = 38块
Bạn ấy: 38 + 3 + 15 = 56块
Tổng: 94块

</details>
