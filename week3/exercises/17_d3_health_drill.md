## Bài Tập 1: 了 Hoàn Thành — Dịch Câu (15 phút)

| # | Tiếng Việt | Tiếng Trung | Đáp án |
|---|-----------|-------------|--------|
| 1 | Tôi đã ăn sáng rồi. | | <details><summary>🔑</summary>我吃了早饭。/ 我吃早饭了。</details> |
| 2 | Anh ấy đã đi rồi. | | <details><summary>🔑</summary>他去了。/ 他走了。</details> |
| 3 | Tôi đã mua 2 cuốn sách. | | <details><summary>🔑</summary>我买了两本书。</details> |
| 4 | Cô ấy đã uống 3 ly cà phê. | | <details><summary>🔑</summary>她喝了三杯咖啡。</details> |
| 5 | Tôi chưa ăn. | | <details><summary>🔑</summary>我没吃。(没+V, bỏ 了!)</details> |
| 6 | Bạn ăn chưa? | | <details><summary>🔑</summary>你吃了吗？/ 你吃了没有？</details> |
| 7 | Tôi ăn xong rồi. | | <details><summary>🔑</summary>我吃完了。</details> |
| 8 | Hôm qua tôi đã đi bệnh viện. | | <details><summary>🔑</summary>昨天我去了医院。</details> |

**Điểm: ___/8**

## Bài Tập 2: 没 + V vs V + 了 — Chọn Đúng (10 phút)

| # | Tình huống | Đáp án đúng | Đáp án |
|---|-----------|------------|--------|
| 1 | Bạn hỏi "ăn chưa?", bạn ĐÃ ăn | A. 吃了 B. 没吃 | <details><summary>🔑</summary>A. 吃了</details> |
| 2 | Bạn chưa uống thuốc | A. 吃了药 B. 没吃药 | <details><summary>🔑</summary>B. 没吃药</details> |
| 3 | Anh ấy đã về nhà rồi | A. 回家了 B. 没回家 | <details><summary>🔑</summary>A. 回家了</details> |
| 4 | Cô ấy chưa đến | A. 来了 B. 没来 | <details><summary>🔑</summary>B. 没来</details> |
| 5 | Tôi đã làm xong bài tập | A. 做完了 B. 没做完 | <details><summary>🔑</summary>A. 做完了</details> |

**Điểm: ___/5**

## Bài Tập 5: Đọc Hiểu — Trả Lời Bằng TQ (10 phút)

```
小明昨天感冒了，头疼，也有一点发烧。
他没去上班，在家休息了一天。
他吃了药，喝了很多水。
今天他好多了，已经去上班了。
```

| # | 问题 | 用中文回答 | Đáp án |
|---|------|---------|--------|
| 1 | 小明昨天怎么了？ | | <details><summary>🔑</summary>他感冒了，头疼，有一点发烧。</details> |
| 2 | 他昨天去上班了吗？ | | <details><summary>🔑</summary>没去。他在家休息了。</details> |
| 3 | 他吃药了吗？ | | <details><summary>🔑</summary>吃了，也喝了很多水。</details> |
| 4 | 今天他怎么样？ | | <details><summary>🔑</summary>好多了，已经去上班了。</details> |
