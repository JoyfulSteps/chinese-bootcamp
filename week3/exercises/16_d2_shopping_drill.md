## Bài Tập 1: 比 — So Sánh (15 phút)

### 1.1 Dịch sang tiếng Trung (dùng 比):

| # | Tiếng Việt | Tiếng Trung | Đáp án |
|---|-----------|-------------|--------|
| 1 | Anh ấy cao hơn tôi. | | <details><summary>🔑</summary>他比我高。</details> |
| 2 | Hôm nay nóng hơn hôm qua. | | <details><summary>🔑</summary>今天比昨天热。</details> |
| 3 | Máy bay nhanh hơn tàu hỏa nhiều. | | <details><summary>🔑</summary>飞机比火车快多了。</details> |
| 4 | Cái này đắt hơn cái kia một chút. | | <details><summary>🔑</summary>这个比那个贵一点。</details> |
| 5 | TQ khó hơn tiếng Anh. | | <details><summary>🔑</summary>中文比英文难。</details> |
| 6 | Anh ấy lớn hơn tôi 3 tuổi. | | <details><summary>🔑</summary>他比我大三岁。</details> |

### 1.2 Sửa câu sai:

| # | Câu sai | Sửa | Đáp án |
|---|---------|-----|--------|
| 1 | 他比我很高。 | | <details><summary>🔑</summary>他比我高。(比 KHÔNG dùng 很!)</details> |
| 2 | 今天比昨天是热。 | | <details><summary>🔑</summary>今天比昨天热。(不用是!)</details> |

**Điểm: ___/8**

## Bài Tập 2: 没有...那么... Phủ Định So Sánh (10 phút)

Viết câu phủ định dùng 没有那么:

| # | Câu so sánh | Phủ định | Đáp án |
|---|------------|---------|--------|
| 1 | 他比我高。(anh cao hơn tôi) | Tôi không cao bằng anh ấy. | <details><summary>🔑</summary>我没有他那么高。</details> |
| 2 | 今天比昨天热。 | Hôm qua không nóng bằng hôm nay. | <details><summary>🔑</summary>昨天没有今天那么热。</details> |
| 3 | 这个比那个贵。 | Cái kia không đắt bằng cái này. | <details><summary>🔑</summary>那个没有这个那么贵。</details> |

**Điểm: ___/3**

## Bài Tập 3: Mua Hàng — Hoàn Thành Hội Thoại (15 phút)

```
顾客: 你好！这___衣服多少___？   (cái áo này bao nhiêu tiền?)
店员: 这件三百___。              (300 đồng)
顾客: ___贵了！那件呢？          (Quá đắt!)
店员: 那件___百块，比这件___一点。 (200, rẻ hơn chút)
顾客: 什么___的？                 (màu gì?)
店员: ___红色的和黑色的。         (có đỏ và đen)
顾客: 我___黑色的。               (muốn màu đen)
店员: 好的，两百块。
顾客: ___！                       (cảm ơn!)
```

<details>
<summary>🔑 Đáp án</summary>

件, 钱 / 块 / 太 / 两, 便宜 / 颜色 / 有 / 要 / 谢谢

</details>

## Bài Tập 5: Đọc Hiểu Không Pinyin (10 phút)

```
昨天我去商店买衣服。商店里有很多衣服。
我看了一件红色的，三百块，太贵了。
旁边有一件黑色的，两百块，比红色的便宜一百块。
黑色的也比红色的好看。

我问店员："可以便宜一点吗？"
店员说："好吧，一百八十块。"
我很高兴，买了那件黑色的衣服。
```

| # | 问题 | 答案 | Đáp án |
|---|------|------|--------|
| 1 | 红色的衣服多少钱？ | | <details><summary>🔑</summary>三百块 (300)</details> |
| 2 | 黑色的比红色的便宜多少？ | | <details><summary>🔑</summary>一百块 (100)</details> |
| 3 | 最后买了什么颜色的？ | | <details><summary>🔑</summary>黑色的 (đen)</details> |
| 4 | 最后花了多少钱？ | | <details><summary>🔑</summary>一百八十块 (180)</details> |

**Điểm: ___/4**
