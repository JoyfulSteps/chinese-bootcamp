## Bài Tập 1: 因为...所以... Nối Câu (15 phút)

### 1.1 Nối nguyên nhân → kết quả:

| Nguyên nhân (因为) | Kết quả (所以) |
|-------------------|---------------|
| A. 下雨了 | 1. 我没买 |
| B. 太贵了 | 2. 我没去上班 |
| C. 他生病了 | 3. 我每天学 |
| D. 我喜欢中文 | 4. 他没来开会 |

<details>
<summary>🔑 Đáp án</summary>
A→2 (mưa → không đi làm), B→1 (đắt → không mua), C→4 (bệnh → không đến họp), D→3 (thích TQ → học mỗi ngày)
</details>

### 1.2 Dịch sang TQ:

| # | Tiếng Việt | Tiếng Trung | Đáp án |
|---|-----------|-------------|--------|
| 1 | Vì trời nóng quá, nên tôi không muốn đi. | | <details><summary>🔑</summary>因为太热了，所以我不想去。</details> |
| 2 | Vì tôi thích viết code, nên tôi làm dev. | | <details><summary>🔑</summary>因为我喜欢写代码，所以我当程序员。</details> |
| 3 | Vì bệnh, nên không đến. | | <details><summary>🔑</summary>因为生病了，所以没来。</details> |
| 4 | Tại sao bạn học TQ? | | <details><summary>🔑</summary>你为什么学中文？</details> |
| 5 | Vì tôi muốn đi TQ làm việc. | | <details><summary>🔑</summary>因为我想去中国工作。</details> |

**Điểm: ___/9**

## Bài Tập 4: Nói Về CÔNG VIỆC CỦA BẠN (10 phút)

Viết 5+ câu:

```
我是______________________________。(nghề)
我在______________________________工作。(nơi)
每天____时上班，____时下班。       (giờ)
因为________________________________,
所以________________________________。(lý do làm nghề này)
我觉得我的工作____________________。
```

- [ ] 🎤 Đã nói to + ghi âm

## Bài Tập 5: Đọc Hiểu (10 phút)

```
小明是程序员。他每天早上九点上班。
他的同事很多，大家经常一起吃午饭。
他为什么当程序员？因为他喜欢电脑。
他觉得写代码很有意思，但是有时候要加班，很累。
他的工资还不错，每个月一万块。
```

| # | 问题 | Đáp án |
|---|------|--------|
| 1 | 小明做什么工作？ | <details><summary>🔑</summary>程序员</details> |
| 2 | 他为什么当程序员？ | <details><summary>🔑</summary>因为他喜欢电脑</details> |
| 3 | 他觉得写代码怎么样？ | <details><summary>🔑</summary>很有意思</details> |
| 4 | 他的工资多少？ | <details><summary>🔑</summary>每个月一万块</details> |
