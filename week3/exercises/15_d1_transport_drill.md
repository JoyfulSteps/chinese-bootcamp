## Bài Tập 3: Đọc Hiểu Không Pinyin (10 phút)

Đọc đoạn văn sau, trả lời câu hỏi:

```
小明家在学校旁边。每天早上他走路去学校，
只要五分钟。他妈妈在医院工作，医院很远，
她每天坐公共汽车去上班，从家到医院要
四十五分钟。他爸爸开车去公司，二十分钟就到了。
```

| # | Câu hỏi | Trả lời | Đáp án |
|---|---------|---------|--------|
| 1 | 小明怎么去学校？ | | <details><summary>🔑</summary>走路去 (đi bộ)</details> |
| 2 | 从家到学校要几分钟？ | | <details><summary>🔑</summary>五分钟 (5 phút)</details> |
| 3 | 妈妈在哪里工作？ | | <details><summary>🔑</summary>在医院 (ở bệnh viện)</details> |
| 4 | 妈妈怎么去上班？ | | <details><summary>🔑</summary>坐公共汽车 (đi xe bus)</details> |
| 5 | 爸爸怎么去公司？ | | <details><summary>🔑</summary>开车去 (lái xe đi)</details> |

**Điểm: ___/5**
