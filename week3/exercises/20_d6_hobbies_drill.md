## Bài Tập 2: 因为所以 vs 虽然但是 — Chọn Đúng (10 phút)

| # | Câu | 因为所以 / 虽然但是 | Đáp án |
|---|-----|-------------------|--------|
| 1 | ___下雨了，___我没去。(mưa → không đi = logic) | | <details><summary>🔑</summary>因为...所以... (nguyên nhân → kết quả)</details> |
| 2 | ___下雨了，___我还是去了。(mưa → vẫn đi = ngược) | | <details><summary>🔑</summary>虽然...但是... (nhượng bộ)</details> |
| 3 | ___很贵，___我没买。(đắt → không mua) | | <details><summary>🔑</summary>因为...所以...</details> |
| 4 | ___很贵，___很好看，我买了。(đắt → nhưng vẫn mua) | | <details><summary>🔑</summary>虽然...但是...</details> |
| 5 | ___他生病了，___没来。 | | <details><summary>🔑</summary>因为...所以...</details> |
| 6 | ___他生病了，___还是来了。 | | <details><summary>🔑</summary>虽然...但是...</details> |

**Điểm: ___/6**

## Bài Tập 5: Đọc Hiểu (10 phút)

```
小红喜欢运动。她最喜欢游泳，每个星期游泳三次。
虽然她的工作很忙，但是她觉得运动很重要。

她也对音乐很感兴趣。她会唱歌，唱得很好。
周末她经常跟朋友一起去唱歌。

她不太喜欢看电影，因为觉得没意思。
但是她喜欢看书，每个月看两三本书。
```

| # | 问题 | Đáp án |
|---|------|--------|
| 1 | 小红最喜欢什么运动？ | <details><summary>🔑</summary>游泳</details> |
| 2 | 她每个星期游泳几次？ | <details><summary>🔑</summary>三次</details> |
| 3 | 她为什么不喜欢看电影？ | <details><summary>🔑</summary>因为觉得没意思</details> |
| 4 | 她每个月看几本书？ | <details><summary>🔑</summary>两三本</details> |
| 5 | 她周末跟朋友做什么？ | <details><summary>🔑</summary>一起去唱歌</details> |

**Điểm: ___/5**
