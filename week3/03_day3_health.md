## 📐 Phần 1: Ngữ Pháp — 了 Hoàn Thành (35 phút)

### ⚠️ 了 là ngữ pháp QUAN TRỌNG NHẤT và KHÓ NHẤT tiếng Trung!

```
了 (le) có 2 chức năng khác nhau. Hôm nay học chức năng 1:

了₁ = HOÀN THÀNH (đã làm xong — đặt SAU động từ)
```

### Pattern: V + 了 = đã V xong

```
我吃了。           Wǒ chī le.             Tôi ĐÃ ăn (rồi).
他去了。           Tā qù le.              Anh ấy ĐÃ đi (rồi).
我买了一本书。     Wǒ mǎi le yì běn shū.  Tôi ĐÃ mua 1 cuốn sách.
她喝了两杯咖啡。   Tā hē le liǎng bēi kāfēi. Cô ấy đã uống 2 ly cà phê.
我昨天看了一个电影。Wǒ zuótiān kàn le yí ge diànyǐng. Hôm qua tôi đã xem 1 phim.
```

### V + 了 + Số + LT + N (cụ thể hóa)

```
⚠️ Khi có 了, thường cần nói CỤ THỂ bao nhiêu:

  我吃了三个苹果。    Tôi đã ăn 3 quả táo.
  他买了两件衣服。    Anh ấy đã mua 2 bộ quần áo.
  我学了一百七十四个词。 Tôi đã học 174 từ! ← bạn đó! 😄
```

### Phủ định: 没(有) + V (BỎ 了!)

```
⚠️ QUAN TRỌNG: Phủ định quá khứ = 没 + V, KHÔNG dùng 不 + V + 了!

  ✅ 我没吃。         Wǒ méi chī.           Tôi CHƯA ăn.
  ✅ 他没去。         Tā méi qù.            Anh ấy chưa đi.
  ✅ 我没买书。       Wǒ méi mǎi shū.       Tôi chưa mua sách.

  ❌ 我不吃了。       (nghĩa khác! = "Tôi KHÔNG ĂN nữa" ← dừng lại)
  ❌ 我没吃了。       (SAI ngữ pháp!)
```

### Câu hỏi: V + 了 + 没有?

```
你吃了没有？       Nǐ chī le méi yǒu?     Bạn ăn chưa?
你去了没有？       Nǐ qù le méi yǒu?      Bạn đi chưa?
→ 吃了。          Chī le.                  Ăn rồi.
→ 没吃。          Méi chī.                 Chưa ăn.

Hoặc dùng 吗:
你吃了吗？         Nǐ chī le ma?           Bạn ăn chưa?
```

> 💡 **Dev tip:** `了` giống `commit()` trong Git — hành động ĐÃ HOÀN THÀNH và không thể undo.
> `没 + V` = hành động chưa commit.
> `V + 了` = action committed ✅

### 了₂ (giới thiệu — sẽ học sâu sau)

```
了₂ đặt ở CUỐI CÂU = tình huống MỚI / thay đổi trạng thái

  下雨了。        Xià yǔ le.         Mưa rồi. (bắt đầu mưa = thay đổi)
  他来了。        Tā lái le.          Anh ấy đến rồi. (mới đến)
  我不想吃了。    Wǒ bù xiǎng chī le. Tôi không muốn ăn NỮA. (thay đổi ý)
  太贵了！        Tài guì le!         Quá đắt! (了 nhấn mạnh)

→ Lưu ý phân biệt: 了₁ SAU VERB (hoàn thành) vs 了₂ CUỐI CÂU (thay đổi)
→ Đôi khi cả 2 xuất hiện: 我吃了饭了。(Tôi đã ăn cơm rồi.)
```

## 📚 Phần 2: Cơ Thể & Sức Khỏe (25 phút)

### Bộ phận cơ thể

```
头 tóu      đầu          肚子 dùzi     bụng
眼睛 yǎnjing mắt         腿 tuǐ        chân (đùi)
耳朵 ěrduo   tai          手 shǒu       tay
嘴 zuǐ       miệng       脚 jiǎo       bàn chân
鼻子 bízi    mũi
```

### Triệu chứng & Bệnh

```
不舒服   bù shūfu     không khỏe
疼/痛    téng/tòng    đau
头疼     tóu téng     đau đầu
肚子疼   dùzi téng    đau bụng
发烧     fā shāo      sốt
咳嗽     ké sou       ho
感冒     gǎnmào      cảm lạnh
生病     shēng bìng   bị bệnh
```

### Mẫu câu khi bị ốm

```
我不舒服。        Wǒ bù shūfu.         Tôi không khỏe.
我头疼。          Wǒ tóu téng.         Tôi đau đầu.
我肚子疼。        Wǒ dùzi téng.        Tôi đau bụng.
我感冒了。        Wǒ gǎnmào le.       Tôi bị cảm rồi. (了₂ = thay đổi)
我发烧了。        Wǒ fā shāo le.      Tôi bị sốt rồi.
我要去医院。      Wǒ yào qù yīyuàn.   Tôi phải đi bệnh viện.
你吃药了吗？      Nǐ chī yào le ma?    Bạn uống thuốc chưa?
吃了。            Chī le.              Uống rồi.
没吃。            Méi chī.             Chưa uống.
```

### Cụm từ & Pattern mới

```
你身体好吗？       Nǐ shēntǐ hǎo ma?       Sức khỏe tốt không?
我头疼，不舒服。   Wǒ tóu téng, bù shūfu.   Tôi đau đầu, không khỏe.
你应该去医院。     Nǐ yīnggāi qù yīyuàn.   Bạn nên đi bệnh viện.
你吃药了吗？       Nǐ chī yào le ma?         Bạn uống thuốc chưa?
我吃完了。         Wǒ chī wán le.            Tôi ăn xong rồi.
这个最好。         Zhège zuì hǎo.            Cái này tốt nhất.
他最忙。           Tā zuì máng.              Anh ấy bận nhất.
很难，但是很有意思。 Hěn nán, dànshì hěn yǒu yìsi. Khó nhưng thú vị.
```

### 📝 最 = Nhất (Superlative)

```
最好    zuì hǎo     tốt nhất
最大    zuì dà      lớn nhất
最快    zuì kuài    nhanh nhất
最贵    zuì guì     đắt nhất
最难    zuì nán     khó nhất

So sánh combo:
  比 = hơn (comparative):       A比B好 (A tốt hơn B)
  最 = nhất (superlative):      A最好 (A tốt nhất)
```

## 🗣️ Phần 5: Hội Thoại — Khám Bệnh (15 phút)

```
A(bệnh nhân): 医生你好，我不舒服。
              Yīshēng nǐ hǎo, wǒ bù shūfu.

B(bác sĩ):   你怎么了？哪里不舒服？
              Nǐ zěnme le? Nǎli bù shūfu?

A: 我头疼，也有一点咳嗽。
   Wǒ tóu téng, yě yǒu yìdiǎn ké sou.

B: 发烧了吗？
   Fā shāo le ma?

A: 有一点，三十七度五。
   Yǒu yìdiǎn, sānshíqī dù wǔ.

B: 你感冒了。应该多喝水，多休息。
   Nǐ gǎnmào le. Yīnggāi duō hē shuǐ, duō xiūxi.

A: 要吃药吗？
   Yào chī yào ma?

B: 要，一天三次，吃了饭以后吃。
   Yào, yì tiān sān cì, chī le fàn yǐhòu chī.

A: 好的，谢谢医生！
```

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 头 tóu (đầu) — 5 nét
```
Giản thể của 頭. 头疼 = đau đầu
Tập viết: 头 头 头 头 头 头 头 头 头 头
```

#### 病 bìng (bệnh) — 10 nét
```
疒(bệnh tật) + 丙 → "nằm bệnh" = BỆNH
Bộ thủ: 疒— liên quan BỆNH TẬT
生病 = bị bệnh, 病人 = bệnh nhân
Tập viết: 病 病 病 病 病 病 病 病 病 病
```

#### 药 yào (thuốc) — 9 nét
```
艹(cỏ) + 约 → "cỏ thuốc" = THUỐC
吃药 = uống thuốc
Tập viết: 药 药 药 药 药 药 药 药 药 药
```

#### 完 wán (xong/hết) — 7 nét
```
宀(mái) + 元(gốc) → "mái che gốc = KẾT THÚC"
吃完 = ăn xong, 做完 = làm xong, 看完 = xem xong
Tập viết: 完 完 完 完 完 完 完 完 完 完
```

#### 最 zuì (nhất) — 12 nét
```
最好 = tốt nhất, 最大 = lớn nhất
Tập viết: 最 最 最 最 最 最 最 最 最 最
```

#### 感 gǎn (cảm) — 13 nét
```
感冒 = cảm lạnh, 感觉 = cảm giác
Tập viết: 感 感 感 感 感 感 感 感 感 感
```

#### 疼 téng (đau) — 10 nét
```
疒(bệnh) + 冬(đông) → "mùa đông bệnh = ĐAU"
头疼 = đau đầu, 肚子疼 = đau bụng
Tập viết: 疼 疼 疼 疼 疼 疼 疼 疼 疼 疼
```

#### 但 dàn (nhưng) — 7 nét
```
亻(người) + 旦(sáng sớm) → "ngày mới NHƯNG..."
但是 = nhưng mà
Tập viết: 但 但 但 但 但 但 但 但 但 但
```
