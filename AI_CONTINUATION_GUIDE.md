## 📐 Format & Quy Tắc Tạo Bài Học

### Cấu trúc mỗi ngày học (2 files):

**1. File bài học** (`NN_dayN_topic.md`):
```
# Day N: [Chủ đề] & [Ngữ pháp]
## Tuần X — Ngày Y (Day N) | [Thứ] [Ngày/Tháng/Năm]

## 🎯 Mục Tiêu Ngày N (bảng)
## 🔄 Ôn Tập Ngày Trước (10 phút, quick check)
## 📐 Phần 1-2: Ngữ Pháp Mới (25-35 phút)
   → Dev tip (so sánh với lập trình)
   → Ví dụ + Thực hành
## 📖 Phần 3: Từ Vựng Mới — 13 từ (bảng có câu ví dụ)
   → Cụm từ & Câu thực hành
## 📖 Phần 4: Bài Đọc Không Pinyin (từ tuần 3+)
   → Câu hỏi + đáp án spoiler
## 🗣️ Phần 5: Hội Thoại + Shadowing
## ✏️ Phần 6: Hán Tự Mới — 8 chữ (phân tích bộ thủ)
## 📋 Checklist Cuối Ngày
## 📂 Navigation (← Trước | Đang học | Tiếp →)
```

**2. File bài tập** (`exercises/NN_dN_topic_drill.md`):
```
# 🏋️ Exercise NN — [Chủ đề] Drill
## Day N (Tuần X, Ngày Y)

## Bài Tập 1-6: (mỗi BT 5-15 phút)
   → Bảng có cột đáp án dùng <details><summary>🔑</summary>...</details>
   → Điểm: ___/N mỗi bài
## 📊 Tổng Kết Exercise (bảng tổng điểm)
## 📂 Navigation
```

### Quy tắc quan trọng:
- **13 từ mới/ngày** (tăng từ 10 ở tuần 1)
- **8 Hán tự mới/ngày** (có phân tích bộ thủ)
- **Đáp án spoiler:** dùng `<details><summary>🔑</summary>nội dung</details>`
- **Dev tips:** so sánh ngữ pháp TQ với khái niệm lập trình
- **Bài đọc không Pinyin:** từ tuần 3 trở đi, tăng dần (60→100→150 chữ)
- **Shadowing:** từ tuần 3, gõ vào Google Translate → nghe → đọc theo
- **Tích lũy từ liên tục:** mỗi file ghi rõ "Tổng tích lũy sau Day N: XXX từ"
- **Ngày 7 mỗi tuần:** Ôn tập + Assessment test + Báo cáo
