## 📐 Phần 1: Ngữ Pháp — 不管...都 (20 phút)

### Pattern: 不管 A 都 B = Bất kể A, đều B

```
不管 bùguǎn = bất kể/mặc kệ     都 dōu = đều

不管多难，我都要学会。
Bất kể khó thế nào, tôi đều phải học.

不管你用什么框架，基础都很重要。
Bất kể dùng FW gì, foundation đều quan trọng.

不管出了什么问题，都要先备份。
Bất kể xảy ra vấn đề gì, đều phải backup trước.
```

### So sánh 三兄弟:
```
只要A就B  = Chỉ cần A thì B (điều kiện đủ)
  只要测试通过，就可以上线。

不管A都B  = Bất kể A, đều B (vô điều kiện)
  不管用什么语言，都要写好代码。

除了A以外  = Ngoài A ra (liệt kê)
  除了Java以外，还会Python。
```

> 💡 **Dev tip:** `只要就` = `if (x) { do(); }` | `不管都` = `finally { always(); }` — finally block luôn chạy!

```

↓ 开头 - giới thiệu (3-4 câu)
```
