## 📖 Phần 1: Đọc GitHub Issues Mẫu (30 phút)

### Issue 1 — Bug Report:
```
🐛 Issue #156: 用户登录后页面白屏

**描述**: 用户登录成功后，页面显示白屏，控制台
报错 "Cannot read property 'name' of undefined"。

**复现步骤**:
1. 打开登录页面
2. 输入正确的用户名和密码
3. 点击登录按钮
4. 页面白屏

**预期结果**: 登录成功后应该跳转到首页。
**实际结果**: 白屏，控制台报错。
**环境**: Chrome 120, macOS

**可能的原因**: 用户信息接口返回的数据结构
发生了变化，前端没有做空值判断。
```

### Issue 2 — Feature Request:
```
💡 Issue #200: 支持暗黑模式

**需求描述**: 希望应用支持暗黑模式，让用户
可以在白天和晚上切换不同的主题。

**具体要求**:
1. 在设置页面添加主题切换按钮
2. 支持浅色和深色两种模式
3. 记住用户的选择

**优先级**: 中
**相关页面**: 所有页面
```

### 词汇:
```
白屏 báipíng = white screen     控制台 kòngzhìtái = console
报错 bàocuò = 报 error          复现 fùxiàn = reproduce
预期 yùqī = expected            跳转 tiàozhuǎn = redirect
可能 kěnéng = possible          暗黑 ànhēi = dark
模式 móshì = mode               切换 qiēhuàn = switch
主题 zhǔtí = theme              优先级 yōuxiānjí = priority
```
