## 📚 Phần 1: Interview Framework TQ (30 phút)

### Part 1 — 自我介绍 (1 phút):
```
面试官好！我叫[名字]，来自越南。
我是一个后端开发，有五年工作经验。
我熟悉Java和Spring Boot框架，
也会Python和JavaScript。

来中国之前，我在越南的一家IT公司工作。
负责电商平台的后端开发。
我的中文学了六周，虽然还不太好，
但是我越来越能用中文交流了。

我的优势是：不但技术基础好，
而且有丰富的实际项目经验。
```

### Part 2 — 项目经验 (1 phút):
```
我最有经验的项目是一个电商平台。用户大概
一百万。我负责订单系统的后端开发。

技术栈是Spring Boot + MySQL + Redis。
我们用微服务架构，把系统拆分成十几个服务。

我做过的最大贡献是优化了查询性能。
之前一个查询要三秒，我优化了SQL和索引以后，
只需要零点三秒。性能提高了十倍。
```

### Part 3 — 行为面试 (1 phút):
```
Q: 你遇到过什么困难？怎么解决的？
A: 有一次上线以后系统出了严重的bug。
   数据库连接池满了，系统不能正常运行。
   我先回滚到上一个版本，然后花了两天
   时间找到原因，是配置文件有错误。
   修改以后重新测试和上线，问题解决了。

Q: 你最大的优势是什么？
A: 我的优势是学习能力强。比如我六周就学会了
   基本的中文。只要给我时间，我就能学会
   任何新技术。
```


## ✏️ Phần 4: 8 Hán Tự Mới (20 phút)

#### 贡 gòng (cống) — 7 nét
```
贡献 = đóng góp/contribute
Tập viết: 贡 贡 贡 贡 贡 贡 贡 贡 贡 贡
```
#### 献 xiàn (hiến) — 13 nét
```
贡献 = đóng góp, 奉献 = cống hiến
Tập viết: 献 献 献 献 献 献 献 献 献 献
```
#### 严 yán (nghiêm) — 7 nét
```
严重 = nghiêm trọng, 严格 = nghiêm khắc
Tập viết: 严 严 严 严 严 严 严 严 严 严
```
#### 困 kùn (khốn) — 7 nét
```
困难 = khó khăn, 困惑 = bối rối
Tập viết: 困 困 困 困 困 困 困 困 困 困
```
#### 难 nán (nan/khó) — 10 nét
```
困难 = khó khăn, 难度 = độ khó
Tập viết: 难 难 难 难 难 难 难 难 难 难
```
#### 强 qiáng (cường) — 12 nét
```
能力强 = năng lực mạnh, 坚强 = kiên cường
Tập viết: 强 强 强 强 强 强 强 强 强 强
```
#### 望 wàng (vọng) — 11 nét
```
希望 = hy vọng, 期望 = kỳ vọng
Tập viết: 望 望 望 望 望 望 望 望 望 望
```
#### 积 jī (tích) — 10 nét
```
积累 = tích lũy, 积极 = tích cực
Tập viết: 积 积 积 积 积 积 积 积 积 积
```
