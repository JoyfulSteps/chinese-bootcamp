## 📚 Phần 1: Pinyin Tổ Hợp Đặc Biệt (30 phút)

### 1.1 Âm tiết đặc biệt: -i "đặc biệt"

```
⚠️ Có 3 loại "i" khác nhau trong Pinyin:

Loại 1: "i" thật (giống tiếng Việt "i")
  Sau: b p m d t n l j q x y
  Ví dụ: bī, pí, mǐ, dì, jī, xī, yī
  → Nguyên âm "i" bình thường

Loại 2: "i" cong lưỡi (chỉ sau zh ch sh r)
  zhi  chi  shi  ri
  → Đọc: giữ lưỡi CONG, thêm âm "ư" rất nhẹ
  → KHÔNG đọc là "zhi = chi" tiếng Việt!
  → Gần đúng: zhư, chư, shư, rư (ư cong lưỡi)

Loại 3: "i" phẳng (chỉ sau z c s)
  zi  ci  si
  → Đọc: giữ lưỡi PHẲNG (sau răng dưới), thêm "ư" nhẹ
  → Gần đúng: dzư, tsư, sư (ư phẳng lưỡi)
```

> 💡 **Dev tip:** `i` giống một **polymorphic type** — cùng ký hiệu "i" nhưng behavior khác tùy context (sau j/q/x → real "i", sau zh/ch/sh/r → retroflex "i", sau z/c/s → flat "i").

**🎯 Luyện phân biệt:**
```
Đọc 5 lần mỗi hàng, chú ý "i" khác nhau:

jī (i thật, cao)     zhī (i cong lưỡi)     zī (i phẳng lưỡi)
qī (i thật, cao)     chī (i cong lưỡi)     cī (i phẳng lưỡi)
xī (i thật, cao)     shī (i cong lưỡi)     sī (i phẳng lưỡi)
```

---

### 1.2 Tổ hợp khó thường gặp

| Tổ hợp | Phát âm | Sai phổ biến | Ví dụ |
|--------|---------|-------------|-------|
| **üe** (sau j/q/x viết "ue") | "ü" + "ê" | Hay đọc thành "uê" | xué 学 (học), jué 觉 |
| **un** (sau j/q/x thực ra = ün) | "ü" + "n" | Hay đọc thành "un" thường | jūn 军 (quân) |
| **iou** (viết "iu") | "i" + "ôu" | Hay quên "o" ở giữa | liù 六 (6), jiǔ 九 (9) |
| **uei** (viết "ui") | "u" + "ây" | Hay quên "e" ở giữa | huí 回 (về), guì 贵 (đắt) |
| **uen** (viết "un") | "u" + "ơn" | Hay quên "e" ở giữa | wén 文, lùn 论 |
| **er** | "ơ" + cuốn lưỡi | Hay đọc thành "ơ" phẳng | èr 二 (hai), ér 儿 (con) |

### 1.3 Nhi hóa (儿化 — er-ization)

```
Ở Bắc Kinh, nhiều từ thêm "r" cuối → gọi là "nhi hóa"

  哪里 nǎli   →  哪儿 nǎr     (ở đâu — informal)
  这里 zhèli  →  这儿 zhèr    (ở đây — informal)  
  那里 nàli   →  那儿 nàr     (ở kia — informal)
  一点 yìdiǎn →  一点儿 yìdiǎnr (một ít — rất phổ biến)

⚠️ Không bắt buộc phải nói nhi hóa, nhưng cần NGHE HIỂU được.
   Người miền Nam TQ ít dùng nhi hóa.
```

## 📐 Phần 2: Ngữ Pháp — 的 (de) Sở Hữu & Mô Tả (30 phút)

### 的 là gì?

```
的 (de) = từ kết nối QUAN TRỌNG NHẤT tiếng Trung
Giống "của" / "'s" nhưng dùng rộng hơn nhiều

3 chức năng chính:
  1. Sở hữu:   A 的 B = B của A
  2. Mô tả:    Adjective 的 Noun = ... Noun
  3. Nominaliztion: Verb 的 = người/vật làm verb đó
```

### Chức năng 1: Sở hữu (của)

```
我的     wǒ de        của tôi
你的     nǐ de        của bạn
他的     tā de        của anh ấy
她的     tā de        của cô ấy
我们的   wǒmen de     của chúng tôi

我的电脑     wǒ de diànnǎo       máy tính của tôi
他的名字     tā de míngzi         tên của anh ấy
中国的文化   Zhōngguó de wénhuà   văn hóa của TQ
```

### Chức năng 2: Mô tả (tính từ + 的 + danh từ)

```
好的人        hǎo de rén          người tốt
大的公司      dà de gōngsī        công ty lớn  
很好的朋友    hěn hǎo de péngyou  bạn rất tốt
新的电脑      xīn de diànnǎo      máy tính mới
```

### ⚠️ Khi nào BỎ 的?

```
Quy tắc: Quan hệ thân thiết / tính từ đơn âm + danh từ → BỎ 的

  ✅ 我妈妈     wǒ māma        mẹ tôi (thân thiết → bỏ 的)
  ✅ 好人       hǎo rén        người tốt (1 âm tiết adj → bỏ 的)
  ✅ 中国人     Zhōngguó rén   người TQ (quốc gia + rén → bỏ 的)

  So sánh:
  我的朋友   wǒ de péngyou   bạn CỦA tôi (giữ 的 — rõ ràng hơn)
  我朋友     wǒ péngyou      bạn tôi (bỏ 的 — informal, nghe tự nhiên)
  → CẢ HAI đều đúng!
```

> 💡 **Dev tip:** `的` giống `.` (dot operator) trong OOP: `wǒ.diànnǎo` = `我的电脑`. Khi context rõ ràng thì có thể sugar syntax (bỏ `.`).

### 🏋️ Thực hành 的

```
Dịch sang tiếng Trung:

1. Máy tính của tôi         → __________________
2. Tên của bạn              → __________________
3. Bạn tốt                  → __________________
4. Sách của anh ấy          → __________________
5. Người Việt Nam (của VN)  → __________________
```

<details>
<summary>🔑 Đáp án</summary>

1. 我的电脑 wǒ de diànnǎo
2. 你的名字 nǐ de míngzi
3. 好的朋友 hǎo de péngyou (hoặc: 好朋友)
4. 他的书 tā de shū
5. 越南人 Yuènán rén (bỏ 的 — quốc gia+人)

</details>

### Windows 10/11

```
1. Settings → Time & Language → Language

5. Chuyển đổi: Win + Space (hoặc Alt + Shift)
```

## ✏️ Phần 5: 8 Hán Tự Mới (30 phút)

#### 书 shū (sách) — 4 nét
```
横折+横+竖+点 → giống quyển sách mở
Tập viết: 书 书 书 书 书 书 书 书 书 书
```

#### 水 shuǐ (nước) — 4 nét
```
Giống dòng nước chảy: nét dọc + các nét phẩy
Bộ thủ: 水 chính là bộ thủ "nước" (氵khi ở bên trái)
Tập viết: 水 水 水 水 水 水 水 水 水 水
```

#### 电 diàn (điện) — 5 nét
```
日 + 乚 (nét cong) → "mặt trời + dây" = ĐIỆN
电脑 = máy tính, 电话 = điện thoại
Tập viết: 电 电 电 电 电 电 电 电 电 电
```

#### 手 shǒu (tay) — 4 nét
```
Hình bàn tay: 3 nét ngang + 1 nét dọc cong
Bộ thủ: 手 → 扌(khi ở trái: đánh, đẩy, kéo...)
Tập viết: 手 手 手 手 手 手 手 手 手 手
```

#### 友 yǒu (bạn) — 4 nét
```
朋友 péngyou = bạn bè
Tập viết: 友 友 友 友 友 友 友 友 友 友
```

#### 老 lǎo (già/cũ) — 6 nét
```
老师 lǎoshī = giáo viên, 老人 lǎorén = người già
Tập viết: 老 老 老 老 老 老 老 老 老 老
```

#### 新 xīn (mới) — 13 nét
```
亲 + 斤 → "gần + rìu" → "chặt cái cũ = MỚI"
Đối lập: 旧 jiù (cũ)
Tập viết: 新 新 新 新 新 新 新 新 新 新
```

#### 茶 chá (trà) — 9 nét
```
艹(cỏ) + 人 + 木(cây) → "cỏ trên cây với người" = TRÀ
Tập viết: 茶 茶 茶 茶 茶 茶 茶 茶 茶 茶
```
