## 📐 Phần 2: Ngữ Pháp — 想 vs 要 (Muốn) (25 phút)

### 想 xiǎng = muốn (nhẹ nhàng, mong muốn)

```
我想喝水。       Wǒ xiǎng hē shuǐ.       Tôi MUỐN uống nước. (nhẹ nhàng)
我想去中国。     Wǒ xiǎng qù Zhōngguó.   Tôi MUỐN đi TQ. (mơ ước)
你想吃什么？     Nǐ xiǎng chī shénme?     Bạn muốn ăn gì? (hỏi ý kiến)
我想学中文。     Wǒ xiǎng xué Zhōngwén.   Tôi muốn học TQ. (nguyện vọng)

Phủ định: 不想 bù xiǎng = không muốn
  我不想去。     Wǒ bù xiǎng qù.         Tôi không muốn đi.
```

### 要 yào = muốn (mạnh mẽ, quyết định) / sẽ / cần

```
我要喝水！       Wǒ yào hē shuǐ!         Tôi MUỐN uống nước! (quyết liệt)
我要去中国。     Wǒ yào qù Zhōngguó.     Tôi SẼ đi TQ. (đã quyết)
你要几个？       Nǐ yào jǐ ge?             Bạn muốn mấy cái?
明天要下雨。     Míngtiān yào xià yǔ.     Ngày mai SẼ mưa.

Phủ định: 不要 bú yào = đừng! (cấm/mệnh lệnh)
  不要去！       Bú yào qù!               ĐỪNG đi!
  不要说话！     Bú yào shuō huà!         ĐỪNG nói chuyện!
```

### So sánh trực tiếp

```
         想                          要
     ─────────                   ─────────
Mức độ:  nhẹ (wish)               mạnh (will/want/need)
Nghĩa:  muốn, mong                muốn, cần, sẽ
Phủ định: 不想 (không muốn)       不要 (đừng! ← mạnh hơn)

Ví dụ:
  我想吃米饭。 (Tôi muốn ăn cơm — lịch sự, mềm mại)
  我要吃米饭！ (Tôi muốn/cần ăn cơm! — quyết liệt, đói lắm)
  
  我不想去。   (Tôi không muốn đi — từ chối nhẹ nhàng)
  不要去！     (ĐỪNG đi! — ra lệnh / cấm)
```

> 💡 **Dev tip:** `想` = `would like to` (soft request). `要` = `must / will` (hard requirement). Giống `optional` vs `required` parameter.

### 🏋️ Thực hành

```
Chọn 想 hay 要:

1. 你___吃什么？(Hỏi ý kiến lịch sự)       → ___
2. 我___喝咖啡！(Rất cần, đang buồn ngủ)    → ___
3. ___说话！(Đừng nói chuyện!)               → 不___
4. 我___学中文。(Mong muốn, nguyện vọng)    → ___
5. 明天我___去公司。(Sẽ đi — đã quyết)      → ___
```

<details>
<summary>🔑 Đáp án</summary>

1. 想 (hỏi ý kiến lịch sự)
2. 要 (cần/muốn mạnh)
3. 不要 (cấm/đừng)
4. 想 (nguyện vọng)
5. 要 (sẽ — kế hoạch)

</details>

## 📐 Phần 3: Ngữ Pháp — 时候 (Khi nào) (15 phút)

### ...的时候 = khi.../lúc...

```
吃饭的时候       chī fàn de shíhou        khi ăn cơm
上班的时候       shàngbān de shíhou       khi đi làm
学中文的时候     xué Zhōngwén de shíhou   khi học TQ
我小的时候       wǒ xiǎo de shíhou        khi tôi còn nhỏ

Ví dụ câu:
  吃饭的时候不要看手机。
  Chī fàn de shíhou bú yào kàn shǒujī.
  Khi ăn cơm đừng nhìn ĐT.

  学中文的时候，我听中文歌。
  Xué Zhōngwén de shíhou, wǒ tīng Zhōngwén gē.
  Khi học TQ, tôi nghe nhạc TQ.
```

### 什么时候 = khi nào / bao giờ

```
你什么时候去？   Nǐ shénme shíhou qù?      Bạn bao giờ đi?
你什么时候来？   Nǐ shénme shíhou lái?      Bạn khi nào đến?
他什么时候回家？ Tā shénme shíhou huí jiā?  Anh ấy bao giờ về nhà?
```

## ✏️ Phần 5: 8 Hán Tự Mới (25 phút)

#### 早 zǎo (sớm/sáng) — 6 nét
```
日(mặt trời) + 十(+) → "mặt trời vừa lên" = SỚM
早上 = buổi sáng, 早饭 = bữa sáng
Tập viết: 早 早 早 早 早 早 早 早 早 早
```

#### 回 huí (về/quay lại) — 6 nét
```
囗 bên ngoài + 口 bên trong → "miệng trong miệng = quay VỀ"
回家 = về nhà, 回来 = trở lại
Tập viết: 回 回 回 回 回 回 回 回 回 回
```

#### 开 kāi (mở/bắt đầu) — 4 nét
```
Chỉ 4 nét! Giản thể của 開
开门 = mở cửa, 开车 = lái xe, 开会 = họp
Tập viết: 开 开 开 开 开 开 开 开 开 开
```

#### 饭 fàn (cơm/bữa ăn) — 7 nét
```
饣(thức ăn) + 反(trái) → ĂN CƠM
Bộ thủ: 饣= 食 (thức ăn) — liên quan ĐỒ ĂN
吃饭 = ăn cơm, 米饭 = cơm
Tập viết: 饭 饭 饭 饭 饭 饭 饭 饭 饭 饭
```

#### 午 wǔ (giữa trưa) — 4 nét
```
上午 = buổi sáng, 中午 = trưa, 下午 = buổi chiều
Tập viết: 午 午 午 午 午 午 午 午 午 午
```

#### 睡 shuì (ngủ) — 13 nét
```
目(mắt) + 垂(rũ) → "mắt rũ xuống" = NGỦ
Bộ thủ: 目 — liên quan MẮT
睡觉 = đi ngủ
Tập viết: 睡 睡 睡 睡 睡 睡 睡 睡 睡 睡
```

#### 休 xiū (nghỉ) — 6 nét
```
亻(người) + 木(cây) → "người dựa vào cây" = NGHỈ
休息 = nghỉ ngơi
Tập viết: 休 休 休 休 休 休 休 休 休 休
```

#### 工 gōng (công) — 3 nét
```
Chỉ 3 nét! Hình dụng cụ lao động
工作 = công việc/làm việc, 工人 = công nhân
Tập viết: 工 工 工 工 工 工 工 工 工 工
```
