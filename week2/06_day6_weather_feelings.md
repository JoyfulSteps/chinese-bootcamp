## 📐 Phần 1: Ngữ Pháp — 很 + Tính Từ (25 phút)

### Quy tắc quan trọng: Tính từ tiếng Trung KHÔNG dùng 是!

```
❌ SỐNG VÀ CHẾT: ĐỪNG bao giờ nói 是+Adj!

  ❌ 他是好。    Tā shì hǎo.     (SAI!!)
  ✅ 他很好。    Tā hěn hǎo.     (Anh ấy rất tốt.)

  ❌ 天气是热。  Tiānqì shì rè.   (SAI!!)
  ✅ 天气很热。  Tiānqì hěn rè.   (Thời tiết rất nóng.)

Quy tắc:
  是 (là) → chỉ dùng với DANH TỪ:     他是老师。(Anh ấy LÀ giáo viên.)
  很 (rất) → dùng với TÍNH TỪ:         他很好。  (Anh ấy rất tốt.)
```

### 很 đôi khi chỉ là "filler" (không mang nghĩa "rất")

```
Khi nói 他很好 thực tế có 2 nghĩa:
  1. Anh ấy rất tốt. (nhấn mạnh 很)
  2. Anh ấy tốt.     (很 chỉ là filler, nói bình thường)

→ Khi muốn THẬT SỰ nhấn mạnh "RẤT": nhấn giọng vào 很
→ Hoặc dùng 很+很: 很很好 ← ít dùng
→ Hoặc dùng 非常 fēicháng: 非常好 = RẤT RẤT tốt
```

> 💡 **Dev tip:** `很` giống `default value` — câu tính từ BẮT BUỘC phải có `很` (hoặc từ tương đương) giống function bắt buộc phải return. Nếu không có `很`, câu nghe dang dở.

### Pattern: 主语 + 很 + 形容词

```
今天很热。       Jīntiān hěn rè.         Hôm nay (rất) nóng.
他很忙。         Tā hěn máng.            Anh ấy (rất) bận.
这本书很好。     Zhè běn shū hěn hǎo.    Cuốn sách này (rất) hay.
我很高兴。       Wǒ hěn gāoxìng.         Tôi (rất) vui.
妈妈很漂亮。     Māma hěn piàoliang.      Mẹ (rất) đẹp.
```

### Phủ định: 不 + Adj (bỏ 很)

```
✅ 今天不热。    Jīntiān bú rè.          Hôm nay không nóng.
✅ 他不忙。      Tā bù máng.             Anh ấy không bận.
✅ 我不高兴。    Wǒ bù gāoxìng.          Tôi không vui.

⚠️ Phủ định → bỏ 很, chỉ dùng 不 + Adj
```

## 📐 Phần 2: Ngữ Pháp — 太...了 (Quá...) (20 phút)

### Cấu trúc: 太 + Adj + 了

```
太 tài = QUÁ (mức độ cao hơn 很)

太好了！       Tài hǎo le!             Quá tốt! / Tuyệt vời!
太热了！       Tài rè le!              Quá nóng!
太贵了！       Tài guì le!             Quá đắt!
太大了！       Tài dà le!              Quá lớn!
太忙了！       Tài máng le!            Quá bận!
太难了！       Tài nán le!             Quá khó!
太累了！       Tài lèi le!             Quá mệt!
```

### So sánh mức độ

```
Thang mức độ (từ nhẹ → mạnh):
  不好      bù hǎo              không tốt
  还好      hái hǎo             vẫn ổn / tạm được
  好        hǎo                 tốt
  很好      hěn hǎo             rất tốt
  非常好    fēicháng hǎo        cực kỳ tốt
  太好了    tài hǎo le          quá tốt! tuyệt!

  不好 < 还好 < 好 < 很好 < 非常好 < 太好了
```

> 💡 **Dev tip:** Giống severity levels trong logging:
> `DEBUG(不好) < INFO(还好) < WARN(好) < ERROR(很好) < CRITICAL(太好了)`

## 📚 Phần 3: Thời Tiết (20 phút)

### Từ vựng thời tiết

| Hán tự | Pinyin | Nghĩa | Câu ví dụ |
|--------|--------|-------|-----------|
| **天气** | tiānqì | thời tiết | 今天天气很好。(Thời tiết hôm nay rất tốt.) |
| **热** | rè | nóng | 今天很热。(Hôm nay rất nóng.) |
| **冷** | lěng | lạnh | 今天很冷。(Hôm nay rất lạnh.) |
| **下雨** | xià yǔ | mưa | 明天要下雨。(Ngày mai sẽ mưa.) |

### Hỏi - Đáp về thời tiết

```
Q: 今天天气怎么样？  Jīntiān tiānqì zěnmeyàng?  Thời tiết hôm nay thế nào?

A: 今天天气很好！    Jīntiān tiānqì hěn hǎo!     Rất tốt!
   今天很热。        Jīntiān hěn rè.              Hôm nay rất nóng.
   今天很冷。        Jīntiān hěn lěng.            Hôm nay rất lạnh.
   今天下雨了。      Jīntiān xià yǔ le.           Hôm nay mưa rồi.
   不太好。          Bú tài hǎo.                  Không tốt lắm.
```

### ⚠️ 怎么样 zěnmeyàng = thế nào (hỏi cảm nhận)

```
天气怎么样？     Tiānqì zěnmeyàng?       Thời tiết thế nào?
你怎么样？       Nǐ zěnmeyàng?           Bạn thế nào?
这本书怎么样？   Zhè běn shū zěnmeyàng?  Cuốn sách này thế nào?
你的工作怎么样？ Nǐ de gōngzuò zěnmeyàng? Công việc thế nào?
```

## 🗣️ Phần 6: Mini Dialogue — Nói Về Thời Tiết & Cảm Xúc (15 phút)

### Hội thoại 1: Thời tiết

```
A: 今天天气怎么样？
   Jīntiān tiānqì zěnmeyàng?

B: 今天很热！你不觉得吗？
   Jīntiān hěn rè! Nǐ bù juéde ma?

A: 是，太热了！我想喝水。
   Shì, tài rè le! Wǒ xiǎng hē shuǐ.

B: 明天可能要下雨。
   Míngtiān kěnéng yào xià yǔ.

A: 太好了！下雨不热。
   Tài hǎo le! Xià yǔ bú rè.
```

### Hội thoại 2: Hỏi thăm

```
A: 你好！你怎么样？
   Nǐ hǎo! Nǐ zěnmeyàng?

B: 我很累。今天工作太忙了。
   Wǒ hěn lèi. Jīntiān gōngzuò tài máng le.

A: 你要不要休息一下？
   Nǐ yào bu yào xiūxi yíxià?

B: 好的。我想喝一杯咖啡。
   Hǎo de. Wǒ xiǎng hē yì bēi kāfēi.

A: 好！我也要喝。
   Hǎo! Wǒ yě yào hē.
```

> 🎤 Đọc 2 hội thoại 3 lần → thử đóng vai A và B.

## ✏️ Phần 7: 8 Hán Tự Mới (25 phút)

#### 气 qì (khí) — 4 nét
```
天气 = thời tiết (trời + khí)
Giản thể của 氣
Tập viết: 气 气 气 气 气 气 气 气 气 气
```

#### 热 rè (nóng) — 10 nét
```
执+灬(lửa dưới) → lửa phía dưới = NÓNG
Bộ thủ: 灬= 火 (lửa) — liên quan NHIỆT
Đối lập: 冷 lěng (lạnh)
Tập viết: 热 热 热 热 热 热 热 热 热 热
```

#### 冷 lěng (lạnh) — 7 nét
```
冫(băng) + 令(lệnh) → "băng ra lệnh" = LẠNH
Bộ thủ: 冫= 冰 (băng) — liên quan LẠNH/ĐÔNG
Tập viết: 冷 冷 冷 冷 冷 冷 冷 冷 冷 冷
```

#### 高 gāo (cao) — 10 nét
```
Hình tháp cao
高兴 = vui mừng (cao hứng!)
Tập viết: 高 高 高 高 高 高 高 高 高 高
```

#### 快 kuài (nhanh) — 7 nét
```
忄(tim) + 夬 → "tim đập nhanh" = NHANH
Đối lập: 慢 màn (chậm)
Tập viết: 快 快 快 快 快 快 快 快 快 快
```

#### 雨 yǔ (mưa) — 8 nét
```
Hình mưa rơi: 一(trời) + 冂(khung) + 4 giọt bên trong
下雨 = mưa (xuống mưa)
Tập viết: 雨 雨 雨 雨 雨 雨 雨 雨 雨 雨
```

#### 忙 máng (bận) — 6 nét
```
忄(tim) + 亡(mất) → "tim mất = BẬN"
Bộ thủ: 忄= 心 (tim) — liên quan CẢM XÚC
Tập viết: 忙 忙 忙 忙 忙 忙 忙 忙 忙 忙
```

#### 难 nán (khó) — 10 nét
```
又(tay) + 隹(chim ngắn) → "tay bắt chim = KHÓ"
太难了！= Quá khó!
Tập viết: 难 难 难 难 难 难 难 难 难 难
```
