## Bài Tập 1: 是 hay 很? — Sửa Câu Sai (10 phút)

Xác định câu ĐÚNG hay SAI, sửa lại nếu sai:

| # | Câu | Đúng/Sai | Sửa lại | Đáp án |
|---|-----|---------|---------|--------|
| 1 | 他是好。 | | | <details><summary>🔑</summary>❌ → 他很好。(很+Adj, không dùng 是)</details> |
| 2 | 她是老师。 | | | <details><summary>🔑</summary>✅ (是+danh từ = đúng!)</details> |
| 3 | 今天天气是热。 | | | <details><summary>🔑</summary>❌ → 今天天气很热。</details> |
| 4 | 这本书很好。 | | | <details><summary>🔑</summary>✅ (很+Adj = đúng!)</details> |
| 5 | 我是累。 | | | <details><summary>🔑</summary>❌ → 我很累。</details> |
| 6 | 他是中国人。 | | | <details><summary>🔑</summary>✅ (是+danh từ = đúng!)</details> |
| 7 | 这个东西是贵。 | | | <details><summary>🔑</summary>❌ → 这个东西很贵。</details> |
| 8 | 天气很好。 | | | <details><summary>🔑</summary>✅</details> |

**Điểm: ___/8**

## Bài Tập 2: 很 / 非常 / 太...了 — Chọn Mức Độ (10 phút)

Chọn từ phù hợp theo ngữ cảnh:

| # | Ngữ cảnh | Điền | Đáp án |
|---|----------|------|--------|
| 1 | Thời tiết bình thường tốt | 天气___好。 | <details><summary>🔑</summary>很 (mức bình thường)</details> |
| 2 | Thời tiết cực kỳ nóng, chịu không nổi | ___热___！ | <details><summary>🔑</summary>太热了！(quá → phàn nàn)</details> |
| 3 | Cuốn sách cực kỳ hay (nhấn mạnh) | 这本书___好！ | <details><summary>🔑</summary>非常 (cực kỳ — nhấn mạnh)</details> |
| 4 | Đồ quá đắt, không muốn mua | ___贵___！ | <details><summary>🔑</summary>太贵了！(quá đắt!)</details> |
| 5 | Hôm nay tôi rất vui | 我___高兴。 | <details><summary>🔑</summary>很 (bình thường)</details> |
| 6 | Nghe tin vui, phản ứng "Tuyệt vời!" | ___好___！ | <details><summary>🔑</summary>太好了！(quá tốt = tuyệt!)</details> |

**Điểm: ___/6**

## Bài Tập 3: Thời Tiết — Hỏi Đáp (10 phút)

### 3.1 Trả lời câu hỏi:

```
Q: 今天天气怎么样？

Trả lời theo tình huống:
```

| # | Tình huống | Trả lời (TQ) | Đáp án |
|---|-----------|-------------|--------|
| 1 | Rất nóng | | <details><summary>🔑</summary>今天很热。/ 太热了！</details> |
| 2 | Rất lạnh | | <details><summary>🔑</summary>今天很冷。/ 太冷了！</details> |
| 3 | Đang mưa | | <details><summary>🔑</summary>今天下雨了。</details> |
| 4 | Rất tốt | | <details><summary>🔑</summary>天气很好！/ 非常好！</details> |
| 5 | Không tốt lắm | | <details><summary>🔑</summary>不太好。/ 天气不好。</details> |

### 3.2 Nói thời tiết HÔM NAY (thật):

```
今天是___月___号，星期___。
天气___________________________。
```
