## Bài Tập 1: Nhận Diện Bộ Thủ (15 phút)

| # | Chữ | Bộ thủ? | Nghĩa bộ thủ | Đáp án |
|---|------|---------|-------------|--------|
| 1 | 吃 | | | <details><summary>🔑</summary>口 (miệng) → liên quan ăn/nói</details> |
| 2 | 她 | | | <details><summary>🔑</summary>女 (nữ) → liên quan phụ nữ</details> |
| 3 | 说 | | | <details><summary>🔑</summary>讠(言 — lời nói) → liên quan ngôn ngữ</details> |

| 6 | 钱 | | | <details><summary>🔑</summary>钅(金 — kim loại) → liên quan kim loại = tiền</details> |
| 7 | 家 | | | <details><summary>🔑</summary>宀 (mái nhà) → liên quan nhà ở</details> |
| 8 | 还 | | | <details><summary>🔑</summary>辶 (đi) → liên quan di chuyển</details> |
| 9 | 做 | | | <details><summary>🔑</summary>亻(人 — người) → liên quan người/hành động</details> |

## Bài Tập 4: 会 / 知道 / 能 — Chọn đúng (10 phút)

| # | Câu | Chọn | Đáp án |
|---|-----|------|--------|
| 1 | 我___说中文。(Tôi biết nói TQ — kỹ năng) | 会/知道/能 | <details><summary>🔑</summary>会 (<b>kỹ năng</b> đã học được)</details> |
| 2 | 我___他的名字。(Tôi biết tên anh ấy — thông tin) | 会/知道/能 | <details><summary>🔑</summary>知道 (<b>thông tin</b> biết được)</details> |
| 3 | 你___来吗？(Bạn có thể đến không? — khả năng) | 会/知道/能 | <details><summary>🔑</summary>能 (<b>khả năng</b>/điều kiện cho phép)</details> |
| 4 | 她不___做饭。(Cô ấy không biết nấu ăn — kỹ năng) | 会/知道/能 | <details><summary>🔑</summary>会 (kỹ năng)</details> |
| 5 | 你___这是什么吗？(Bạn biết đây là gì không? — info) | 会/知道/能 | <details><summary>🔑</summary>知道 (thông tin)</details> |
| 6 | 我今天不___去。(Hôm nay tôi không thể đi — khả năng) | 会/知道/能 | <details><summary>🔑</summary>能 (khả năng/điều kiện)</details> |

**Điểm: ___/6**
