## Bài Tập 2: 想 hay 要? (15 phút)

| # | Câu | 想/要 | Lý do | Đáp án |
|---|-----|------|-------|--------|
| 1 | 你___吃什么？(lịch sự hỏi ý kiến) | | | <details><summary>🔑</summary>想 (hỏi nhẹ nhàng)</details> |
| 2 | 我___喝水！(đang khát, cần ngay) | | | <details><summary>🔑</summary>要 (cần/muốn mạnh)</details> |
| 3 | ___说话！(Đừng nói chuyện!) | 不___ | | <details><summary>🔑</summary>不要 (cấm/mệnh lệnh)</details> |
| 4 | 我___去中国。(mơ ước, nguyện vọng) | | | <details><summary>🔑</summary>想 (nguyện vọng)</details> |
| 5 | 明天我___上班。(ngày mai sẽ đi) | | | <details><summary>🔑</summary>要 (sẽ/kế hoạch)</details> |
| 6 | 我不___去了。(từ chối nhẹ nhàng) | | | <details><summary>🔑</summary>不想 (không muốn — mềm)</details> |
| 7 | 你___几个？(Bạn muốn mấy cái? — mua hàng) | | | <details><summary>🔑</summary>要 (yêu cầu cụ thể)</details> |
| 8 | ___看手机！(Đừng nhìn ĐT!) | 不___ | | <details><summary>🔑</summary>不要 (cấm)</details> |

**Điểm: ___/8**
