## Bài Tập 4: 买 mǎi vs 卖 mài (10 phút)

Chọn đúng: **买 (mua)** hay **卖 (bán)**?

| # | Câu | 买/卖 | Đáp án |
|---|-----|------|--------|
| 1 | 我要___这本书。(Tôi muốn mua cuốn này.) | | <details><summary>🔑</summary>买 mǎi (MUA — thanh 3)</details> |
| 2 | 这个商店___什么？(Cửa hàng này bán gì?) | | <details><summary>🔑</summary>卖 mài (BÁN — thanh 4)</details> |
| 3 | 我想去商店___东西。(Tôi muốn đi CH mua đồ.) | | <details><summary>🔑</summary>买 mǎi</details> |
| 4 | 他___电脑。(Anh ấy bán máy tính.) | | <details><summary>🔑</summary>卖 mài</details> |
| 5 | 这个东西多少钱？我要___！(Bao nhiêu? Tôi muốn mua!) | | <details><summary>🔑</summary>买 mǎi</details> |

**Điểm: ___/5**

## Bài Tập 6: Sắp Xếp Câu + Mini Hội Thoại (10 phút)

```
A: ___的___在哪里？ (___của bạn ở đâu?)

A: ___呢？          (Còn ___ đâu?)
B: 在___。          (Ở ___.)
A: 谢谢！
```
