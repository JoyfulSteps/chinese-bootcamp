## Bài Tập 1: Phân Biệt 3 Loại "i" (10 phút)

Đọc to từng hàng, chú ý sự khác biệt:

```
Hàng 1 (i thật — lưỡi trước):    jī  qī  xī  bī  dī  nī   (5 lần)
Hàng 2 (i cong lưỡi):            zhī chī shī rī             (5 lần)
Hàng 3 (i phẳng lưỡi):           zī  cī  sī                 (5 lần)
```

Quiz: Mỗi từ sau, "i" thuộc loại nào?

| # | Từ | Loại i? | Đáp án |
|---|-----|---------|--------|
| 1 | jī 鸡 (gà) | | <details><summary>🔑</summary>i thật (sau j)</details> |
| 2 | shī 诗 (thơ) | | <details><summary>🔑</summary>i cong lưỡi (sau sh)</details> |
| 3 | sī 丝 (tơ) | | <details><summary>🔑</summary>i phẳng lưỡi (sau s)</details> |
| 4 | lǐ 里 (trong) | | <details><summary>🔑</summary>i thật (sau l)</details> |
| 5 | chī 吃 (ăn) | | <details><summary>🔑</summary>i cong lưỡi (sau ch)</details> |
| 6 | zì 字 (chữ) | | <details><summary>🔑</summary>i phẳng lưỡi (sau z)</details> |

**Điểm: ___/6**

## Bài Tập 2: Tổ Hợp Khó — Đọc Đúng (10 phút)

Đọc to, chú ý:

```
üe: xué (học) — jué (cảm giác) — yuè (tháng)
iu: liù (6) — jiǔ (9) — niú (bò)
ui: huí (về) — duì (đúng) — guì (đắt)
er: èr (2) — ér (con) — nǎr (ở đâu?)
```

Viết Pinyin đúng cho:

| # | Mô tả | Pinyin đúng | Đáp án |
|---|-------|-------------|--------|
| 1 | j + üé (học) | | <details><summary>🔑</summary>xué (nhớ: j/q/x + ü → viết u)</details> |
| 2 | l + iù (sáu) | | <details><summary>🔑</summary>liù (iu = iou rút gọn)</details> |
| 3 | ở đâu (nhi hóa) | | <details><summary>🔑</summary>nǎr (哪儿)</details> |
| 4 | ở đây (nhi hóa) | | <details><summary>🔑</summary>zhèr (这儿)</details> |

## Bài Tập 5: Gõ Pinyin — Thực Hành (10 phút)

> Mở Notepad / VS Code → chuyển sang Pinyin input → gõ:

```
Gõ 5 câu sau bằng Pinyin input:

1. 你好！我叫___。
2. 我的电脑很好。
3. 这个东西多少钱？
4. 你会说中文吗？
5. 我会一点，谢谢！

[ ] Đã gõ xong cả 5 câu
```
