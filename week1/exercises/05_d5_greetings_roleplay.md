## Bài Tập 1: Hoàn Thành Hội Thoại (10 phút)

Điền từ còn thiếu:

```
A: ___好！         (Xin chào!)
B: 你好！你___什么名字？  (Bạn tên gì?)
A: 我___小明。你___？     (Tôi tên Tiểu Minh. Còn bạn?)
B: 我叫大卫。你是___国人？ (Bạn người nước nào?)
A: 我___越南人。          (Tôi là người VN.)
B: 认识你很___！          (Rất vui biết bạn!)
A: 我___很高兴！___！     (Tôi cũng rất vui! Tạm biệt!)
```

<details>
<summary>🔑 Đáp án</summary>

你好！/ 叫 / 叫, 呢 / 哪 / 是 / 高兴 / 也, 再见

</details>

## Bài Tập 4: Ghép Cặp Giao Tiếp (10 phút)

Nối câu hỏi với câu trả lời phù hợp:

| # | Câu hỏi | → | Trả lời |
|---|---------|---|---------|
| 1 | 你好吗？ | | A. 我叫小明。 |
| 2 | 你叫什么？ | | B. 我是越南人。 |
| 3 | 你是哪国人？ | | C. 不客气。 |
| 4 | 谢谢！ | | D. 我很好，谢谢！ |
| 5 | 对不起。 | | E. 没关系。 |

<details>
<summary>🔑 Đáp án</summary>

1→D, 2→A, 3→B, 4→C, 5→E

</details>

## Bài Tập 5: Roleplay — Viết Hội Thoại (15 phút)

> Viết 1 đoạn hội thoại gặp gỡ giữa BẠN (lập trình viên VN) và một đồng nghiệp TQ:

```
你: ___________________________________________
同事: ___________________________________________
你: ___________________________________________
同事: ___________________________________________
你: ___________________________________________
同事: ___________________________________________
```

<details>
<summary>💡 Gợi ý mẫu</summary>

```
你: 你好！我叫[Tên]。
同事: 你好！我叫李华。你是哪国人？
你: 我是越南人。你呢？
同事: 我是中国人。你做什么工作？
你: 我是程序员。我在学中文。
同事: 很好！认识你很高兴！
```

</details>
