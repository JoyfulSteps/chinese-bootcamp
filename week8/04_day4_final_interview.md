## 📚 Phần 1: Final Interview Script (30 phút)

### 完整面试 — 3 phút, 4 phần:

**Part 1 — 自我介绍 (45 giây):**
```
面试官好！我叫___，来自越南，是一个后端程序员。
我有___年工作经验，熟悉Java、Spring Boot和MySQL。
我来中国工作是因为中国的技术发展越来越快。
我学了八周中文，不但能看懂技术文档，
而且能用中文讨论技术方案。
```

**Part 2 — 项目经验 (45 giây):**
```
我最有经验的项目是___的后端开发。
技术栈是___，用___架构。
我最大的贡献是优化了___的性能，
从___提高到___，提升了___倍。
只要配置合理，系统就能稳定运行。
```

**Part 3 — 困难和解决 (45 giây):**
```
有一次Production出了严重的bug，___。
我一边排查日志一边讨论方案。
最终发现原因是___，修改以后重新测试。
不管遇到什么困难，我都会冷静地解决。
这次经历让我学会了___。
```

**Part 4 — 你的优势和希望 (45 giây):**
```
对我来说，最大的优势是学习能力强。
只要给我时间，任何新技术我都能掌握。
另外，我善于沟通，能和团队高效合作。
我希望在中国积累更多实际经验，
持续提升自己的技术能力。谢谢！
```
