## 📐 Phần 1: Ngữ Pháp — 一边...一边 (20 phút)

### Pattern: 一边 A 一边 B = Vừa A vừa B (2 việc cùng lúc)

```
一边 yìbiān...一边 yìbiān = vừa...vừa...

我一边听音乐一边写代码。
Tôi vừa nghe nhạc vừa viết code.

一边吃饭一边看技术视频。
Vừa ăn vừa xem video tech.

他一边工作一边学中文。
Anh ấy vừa đi làm vừa học TQ.
```

### Tech examples:
```
一边调试一边看日志。       Vừa debug vừa xem log.
一边开会一边记笔记。       Vừa họp vừa ghi chú.
一边学技术一边做项目。     Vừa học tech vừa làm project.
我们要一边开发一边测试。   Phải vừa dev vừa test.
```

> 💡 **Dev tip:** `一边...一边` = `Promise.all([taskA, taskB])` — chạy song song! Hoặc `multithreading` — 2 thread chạy cùng lúc.

## 📖 Phần 2: Đọc Blog CSDN Mẫu (30 phút)

CSDN = nền tảng blog tech lâu đời nhất TQ

### Bài Đọc — 后端开发常见面试题 (230 chữ)
```
我做后端开发已经五年了。最近很多朋友问我
面试的经验，今天来分享一下。

后端面试一般分为几个部分：
第一是基础知识。面试官会问你关于数据结构和
算法的问题。虽然工作中不一定每天都用，
但是面试一定会考。建议一边刷题一边复习基础。

第二是数据库。你需要了解SQL查询优化、索引、
事务等概念。面试官经常问："你怎么优化一个
很慢的查询？"

第三是系统设计。对于有经验的开发来说，
这部分越来越重要。面试官可能会问你怎么设计
一个高并发的系统。

第四是项目经验。面试官想了解你做过什么项目，
遇到过什么问题，是怎么解决的。建议提前准备
两三个项目的故事。

除了技术问题以外，沟通能力也很重要。面试的
时候，要清楚地表达自己的想法。不但要说出
你用了什么技术，而且要说为什么选择这个技术。

最后，面试是一个越来越需要准备的过程。
建议每个人都准备一份中英文的自我介绍。
对于想在中国找工作的人来说，中文面试能力
是一个很大的优势。
```

## 🗣️ Phần 4: Hội Thoại — Mock Interview TQ (15 phút)
```
面试官: 请先自我介绍一下。
你:     大家好，我叫___，来自越南。我是一个程序员，
        有五年工作经验。我会Java和Python。

面试官: 你做过什么项目？
你:     我做过一个电商网站的后端开发。用的是
        Spring Boot框架，数据库是MySQL。

面试官: 你遇到过什么困难？
你:     有一次数据库查询很慢。我优化了SQL，
        添加了索引，性能提高了十倍。

面试官: 你为什么想来中国工作？
你:     因为中国的技术发展越来越快，我想学习更多。
        而且我一直在学中文，希望有机会用中文工作。

面试官: 你的中文怎么样？
你:     我学了五个星期，能看懂简单的技术文档。
        我会一边工作一边继续学习。
```
> 🎤 **练习面试** 3 次！录音并对比。
