## 📚 Phần 1: Workplace Communication (30 phút)

### Tình huống thường gặp:

**1. Xin nghỉ phép:**
```
你好，经理。我明天想请假一天，因为有事。
可以吗？谢谢。
Xin chào manager. Ngày mai tôi muốn xin nghỉ 1 ngày, có việc.
Được không? Cảm ơn.
```

**2. Báo cáo tiến độ:**
```
经理好。关于这个项目，我来汇报一下进度：
第一个功能已经完成了，正在做第二个。
预计后天可以全部完成。
Manager, về project này tôi báo cáo tiến độ:
Feature 1 xong rồi, đang làm feature 2.
Dự kiến ngày kia hoàn thành hết.
```

**3. Nhờ giúp đỡ:**
```
你好，能不能帮我看一下这段代码？
我觉得这里有问题，但是找不到原因。
Bạn ơi, giúp tôi xem đoạn code này được không?
Tôi thấy chỗ này có vấn đề mà tìm không ra nguyên nhân.
```

**4. Trong cuộc họp:**
```
我有一个问题/建议。
Tôi có 1 câu hỏi/đề nghị.

我同意/不太同意这个方案。
Tôi đồng ý/không hẳn đồng ý phương án này.

我觉得我们可以这样做...
Tôi nghĩ chúng ta có thể làm thế này...
```

## 📧 Phần 2: Viết Email Tiếng Trung (30 phút)

### Template 1 — Xin nghỉ phép:
```
主题：请假申请 — [你的名字]

[经理名字]，您好！

我想请假[日期]，共[天数]天。
原因是[原因]。

我已经把手上的任务安排好了：
1. [任务1] — 已经完成
2. [任务2] — 交给[同事名字]帮忙跟进

如果有问题可以打我电话。
谢谢！

[你的名字]
```

### Template 2 — Báo cáo tiến độ:
```
主题：项目周报 — [项目名]

大家好，

本周工作进度汇报：
1. ✅ [已完成的任务]
2. 🔄 [正在做的任务]，预计[时间]完成
3. ⏳ [下周计划]

遇到的问题：
- [问题描述 + 解决方案/需要帮助]

有任何问题请随时联系我。
谢谢！
```

### Template 3 — 请求 Code Review:
```
主题：请帮忙Code Review — [功能名]

[同事名字]，你好！

我完成了[功能名]的开发，代码已经提交到
[分支名]分支。请帮忙review一下。

主要修改：
1. [修改1]
2. [修改2]

请注意：[需要特别注意的地方]

谢谢！
```

## ✏️ Phần 5: 8 Hán Tự Mới (20 phút)

#### 假 jiǎ/jià (giả/nghỉ) — 11 nét
```
请假 = xin nghỉ, 假期 = kỳ nghỉ, 放假 = nghỉ
Tập viết: 假 假 假 假 假 假 假 假 假 假
```
#### 报 bào (báo) — 7 nét
```
汇报 = báo cáo, 报告 = báo cáo, 报名 = đăng ký
Tập viết: 报 报 报 报 报 报 报 报 报 报
```
#### 排 pái (xếp) — 11 nét
```
安排 = sắp xếp, 排队 = xếp hàng
Tập viết: 排 排 排 排 排 排 排 排 排 排
```
#### 案 àn (án/phương) — 10 nét
```
方案 = phương án, 答案 = đáp án
Tập viết: 案 案 案 案 案 案 案 案 案 案
```
#### 论 lùn (luận) — 6 nét
```
讨论 = thảo luận, 结论 = kết luận
Tập viết: 论 论 论 论 论 论 论 论 论 论
```
#### 因 yīn (nhân/vì) — 6 nét
```
原因 = nguyên nhân, 因为 = vì
Tập viết: 因 因 因 因 因 因 因 因 因 因
```
#### 确 què (xác) — 12 nét
```
确认 = xác nhận, 正确 = chính xác
Tập viết: 确 确 确 确 确 确 确 确 确 确
```
#### 通 tōng (thông) — 10 nét
```
通知 = thông báo, 沟通 = giao tiếp, 交通 = giao thông
Tập viết: 通 通 通 通 通 通 通 通 通 通
```
