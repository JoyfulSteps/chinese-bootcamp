## ✍️ Phần 2: Viết Tech Blog 150+ Chữ (45 phút)

### Hướng Dẫn Viết Blog TQ:

```
结构 (cấu trúc):
1. 标题 (tiêu đề) — ngắn gọn, hấp dẫn
2. 开头 (mở đầu) — giới thiệu chủ đề + tại sao
3. 正文 (phần chính) — nội dung chi tiết + ví dụ
4. 总结 (tổng kết) — kết luận + lời khuyên
```

### Chọn 1 trong 3 đề:

**Đề 1:** 我学中文的经历
```
提示: 什么时候开始？怎么学？进步怎么样？
建议是什么？
```

**Đề 2:** 我最喜欢的编程语言
```
提示: 是什么？为什么？有什么优势？
和其他语言比怎么样？
```

**Đề 3:** 给新人程序员的建议
```
提示: 学什么？怎么学？注意什么？
你的经验是什么？
```

## ✏️ Phần 3: 8 Hán Tự Mới (20 phút)

#### 容 róng (dung) — 10 nét
```
内容 = nội dung, 容易 = dễ dàng
Tập viết: 容 容 容 容 容 容 容 容 容 容
```
#### 结 jié (kết) — 9 nét
```
总结 = tổng kết, 结果 = kết quả
Tập viết: 结 结 结 结 结 结 结 结 结 结
```
#### 历 lì (lịch) — 4 nét
```
经历 = trải nghiệm, 历史 = lịch sử
Tập viết: 历 历 历 历 历 历 历 历 历 历
```
#### 标 biāo (tiêu) — 9 nét
```
目标 = mục tiêu, 标题 = tiêu đề
Tập viết: 标 标 标 标 标 标 标 标 标 标
```
#### 果 guǒ (quả) — 8 nét
```
结果 = kết quả, 如果 = nếu
Tập viết: 果 果 果 果 果 果 果 果 果 果
```
#### 展 zhǎn (triển) — 10 nét
```
发展 = phát triển, 展示 = trình bày
Tập viết: 展 展 展 展 展 展 展 展 展 展
```
#### 量 liàng (lượng) — 12 nét
```
质量 = chất lượng, 数量 = số lượng
Tập viết: 量 量 量 量 量 量 量 量 量 量
```
#### 终 zhōng (chung) — 8 nét
```
最终 = cuối cùng, 终于 = cuối cùng/rốt cuộc
Tập viết: 终 终 终 终 终 终 终 终 终 终
```
