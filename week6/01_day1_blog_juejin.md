## 📐 Phần 1: Ngữ Pháp — 越来越 (20 phút)

### Pattern: 越来越 + Adj/V = Ngày càng...

```
越来越 yuè lái yuè = ngày càng, mỗi lúc một

我的中文越来越好了。
Tiếng Trung của tôi ngày càng tốt.

天气越来越热了。
Thời tiết ngày càng nóng.

学的东西越来越多。
Học ngày càng nhiều thứ.
```

### Tech examples:
```
代码越来越复杂了。     Code ngày càng phức tạp.
用户越来越多。         User ngày càng nhiều.
系统越来越慢了。       System ngày càng chậm.
技术越来越先进。       Công nghệ ngày càng tiên tiến.
Bug越来越难找了。      Bug ngày càng khó tìm.
```

### Mở rộng: 越 A 越 B = Càng A càng B
```
越学越有意思。         Càng học càng thú vị.
越优化越快。           Càng optimize càng nhanh.
代码越改越乱。         Code càng sửa càng loạn.
```

> 💡 **Dev tip:** `越来越` = `trending graph 📈📉`. Dùng khi mô tả xu hướng: performance 越来越好 = graph đi lên!

### Bài Đọc Mẫu — 前端开发的学习路线 (220 chữ)

```
作为一个前端开发，我经常被新人问到："前端应该
怎么学？学习路线是什么？"今天我来分享一下
我的经验。

首先，HTML、CSS和JavaScript是基础。不管用什么
框架，这三个都必须学好。我建议先花两个月
把基础打好。

然后，选择一个前端框架学习。现在最流行的是
React和Vue。对于新人来说，Vue比较简单，
适合入门。React的生态更丰富，大公司用得多。

接下来是工程化工具。比如Webpack、Vite等。
这些工具可以帮助我们优化项目的性能。
随着项目越来越大，工程化越来越重要。

除了技术以外，还有一些软技能很重要：
- 代码审查能力：不但要写好代码，而且要能
  看懂别人的代码
- 文档能力：好的文档可以提高团队效率
- 沟通能力：和后端、产品经理交流很重要

最后，学习是一个持续的过程。技术更新越来越快，
我们要一边工作一边学习，才能跟上时代。

希望这些经验对大家有帮助！
```

🔍 Hán-Việt: 基础(cơ sở)=foundation, 工程化(công trình hóa)=engineering
