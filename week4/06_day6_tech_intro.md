## 🔄 Ôn Tập Tổng Tuần 4 (30 phút)

### 6 Ngữ Pháp Mới Tuần 4:

```
┌────────────────────────────────────────────────────────────┐
│  #19  正在/在 + V + 呢 = đang                              │
│       我正在写代码呢。 / 他在看书呢。                        │
│       Phủ định: 没(在) + V  ← KHÔNG dùng 不正在!            │
├────────────────────────────────────────────────────────────┤
│  #20  V + 过 = đã từng (trải nghiệm)                       │
│       我去过中国。/ 我没吃过北京烤鸭。                       │
│       ⚠️ 了 = cụ thể, 过 = trải nghiệm                    │
├────────────────────────────────────────────────────────────┤
│  #21  能/可以/会 = 3 kiểu "có thể"                          │
│       能 = khả năng, 可以 = phép, 会 = kỹ năng/sẽ           │
├────────────────────────────────────────────────────────────┤
│  #22  把 + O + V + kết quả                                  │
│       把门关上。把作业做完了。                               │
│       ⚠️ O phải cụ thể, V phải có kết quả                  │
├────────────────────────────────────────────────────────────┤
│  #23  被 + (ai) + V = bị/được (bị động)                     │
│       我的手机被偷了。Bug被修好了。                          │
├────────────────────────────────────────────────────────────┤
│  #24  是...的 (nhấn mạnh chi tiết)                          │
│       你是什么时候来的？我是去年来的。                       │
│       你是怎么来的？我是坐飞机来的。                         │
└────────────────────────────────────────────────────────────┘
```

### Speed Review— Dịch nhanh (5 phút):

```
[ ] Tôi đang viết code (正在)  → 我正在写代码呢。
[ ] Tôi từng đi TQ (过)       → 我去过中国。
[ ] Tôi biết nói TQ (会)      → 我会说中文。
[ ] Đóng cửa lại (把)         → 把门关上。
[ ] ĐT bị trộm (被)           → 手机被偷了。
[ ] Năm ngoái đến (是的)       → 我是去年来的。
```

📊 Tuần 5-6 bạn sẽ đọc blog tech TQ (Juejin, CSDN)

### Cụm từ Tech thực tế:

```
我是一个程序员，每天写代码。
Tôi là dev, mỗi ngày viết code.

这个软件有一个bug，需要修一下。
Software này có bug, cần fix.

我们用React框架写前端。
Chúng tôi dùng React framework viết frontend.

数据库里的数据被删了。
Data trong DB bị xóa rồi.

把代码部署到服务器上。
Deploy code lên server.

请帮我测试一下这个接口。
Giúp test API này nhé.

这个网站是用Vue写的。
Website này là viết bằng Vue.
```

### Dev-to-Chinese Quick Map:

```
code       → 代码 dàimǎ
bug        → 错误 cuòwù / BUG (thường dùng)
fix        → 修 xiū / 修好 xiūhǎo
deploy     → 部署 bùshǔ
test       → 测试 cèshì
API        → 接口 jiēkǒu
database   → 数据库 shùjùkù
server     → 服务器 fúwùqì
frontend   → 前端 qiánduān
backend    → 后端 hòuduān
framework  → 框架 kuàngjià
fullstack  → 全栈 quánzhàn
```

> 💡 **Dev tip:** Nhiều thuật ngữ IT trong TQ rất logic:
> - 电脑 = 电(điện) + 脑(não) = "não điện" = máy tính
> - 软件 = 软(mềm) + 件(thứ) = "thứ mềm" = phần mềm  
> - 硬件 = 硬(cứng) + 件(thứ) = "thứ cứng" = phần cứng
> - 数据库 = 数据(dữ liệu) + 库(kho) = "kho dữ liệu" = database

### Bài Đọc 6 — 程序员的一天 (150 chữ)

```
我是一个全栈程序员，在一家互联网公司工作。
我每天早上九点到公司，先打开电脑，看一下
邮件和消息。

数据库是MySQL。我负责后端的接口开发。

下午通常会开会，讨论新的功能和bug。如果有
紧急的bug，我们要马上修。上次服务器被攻击了，

每个星期五，我们把新的代码部署到服务器上。

每次看到自己开发的软件被很多人使用，
我都很开心。这就是我为什么喜欢当程序员。

```

   Nǐ zuò shénme gōngzuò?

B: 我是程序员，在一家互联网公司工作。
   Wǒ shì chéngxùyuán, zài yì jiā hùliánwǎng gōngsī gōngzuò.

   Nǐ xiě shénme yǔyán de dàimǎ?

B: 主要写Java，用Spring框架。也会写一点JavaScript。
   Zhǔyào xiě Java, yòng Spring kuàngjià. Yě huì xiě yìdiǎn JavaScript.

A: 你们的项目是做什么的？
   Nǐmen de xiàngmù shì zuò shénme de?

B: 我们在开发一个网站，是一个电商平台。
   Wǒmen zài kāifā yí ge wǎngzhàn, shì yí ge diànshāng píngtái.

A: 数据库用的是什么？
   Shùjùkù yòng de shì shénme?

B: 用MySQL和Redis。数据量很大，所以我们也用了
   分布式的服务器。
   Yòng MySQL hé Redis. Shùjù liàng hěn dà, suǒyǐ wǒmen yě yòng le
   fēnbùshì de fúwùqì.

A: 听起来很厉害！
   Tīng qǐlái hěn lìhai!

## ✏️ Phần 5: 8 Hán Tự Mới — Tech Edition (20 phút)

#### 软 ruǎn (mềm) — 8 nét
```
车(xe) + 欠(thiếu) → giản thể = MỀM
软件 = phần mềm, 软件工程 = software engineering
Tập viết: 软 软 软 软 软 软 软 软 软 软
```

#### 网 wǎng (lưới/mạng) — 6 nét
```
网 giản thể của 網
网站 = website, 网络 = mạng, 互联网 = internet
Tập viết: 网 网 网 网 网 网 网 网 网 网
```

#### 数 shù/shǔ (số/đếm) — 13 nét
```
数据 = data, 数学 = toán học, 数字 = con số
Tập viết: 数 数 数 数 数 数 数 数 数 数
```

#### 据 jù (cứ/theo) — 11 nét
```
扌(tay) + 居(ở) → "tay nắm giữ = CĂN CỨ"
数据 = dữ liệu, 根据 = căn cứ
Tập viết: 据 据 据 据 据 据 据 据 据 据
```

#### 框 kuàng (khung) — 10 nét
```
木(cây) + 匡(khuông) → "khung gỗ" = KHUNG
框架 = framework
Tập viết: 框 框 框 框 框 框 框 框 框 框
```

#### 测 cè (đo/thử) — 9 nét
```
氵(nước) + 则(quy tắc) → "đo nước" = KIỂM TRA
测试 = test, 测量 = đo lường
Tập viết: 测 测 测 测 测 测 测 测 测 测
```

#### 部 bù (bộ/phần) — 10 nét
```
部署 = deploy, 部门 = phòng ban, 全部 = toàn bộ
Tập viết: 部 部 部 部 部 部 部 部 部 部
```

#### 开 kāi (mở) — 4 nét
```
开发 = phát triển/dev, 开会 = họp, 打开 = mở ra
开源 = open source!
Tập viết: 开 开 开 开 开 开 开 开 开 开
```
