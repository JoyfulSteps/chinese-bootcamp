## 📐 Phần 1: Ngữ Pháp — 正在 (Đang Làm Gì) (30 phút)

### Pattern: 正在 + V (+ 呢)

```
正在 zhèngzài = đang (chính xác tại thời điểm này)
在   zài      = đang (ngắn gọn hơn, dùng nhiều hơn)
呢   ne       = trợ từ cuối câu, thêm sắc thái "đang ~"

3 cách nói "đang":
  ① 正在 + V + 呢  (đầy đủ nhất)
  ② 在 + V + 呢    (phổ biến nhất)
  ③ V + 呢        (ngắn gọn nhất)
```

### Ví dụ:

```
① 我正在学中文呢。   Wǒ zhèngzài xué Zhōngwén ne.
   Tôi ĐANG học tiếng Trung (ngay lúc này).

② 他在看书呢。       Tā zài kàn shū ne.
   Anh ấy đang đọc sách.

③ 你做什么呢？       Nǐ zuò shénme ne?
   Bạn đang làm gì vậy?

   我写代码呢。       Wǒ xiě dàimǎ ne.
   Tôi đang viết code.
```

### Phủ định: 没(在) + V

```
⚠️ KHÔNG dùng 不正在 — SAI!

✅ 没(在) + V:
   他没在看书。       Tā méi zài kàn shū.
   Anh ấy không đang đọc sách.

   我没在睡觉。       Wǒ méi zài shuì jiào.
   Tôi không đang ngủ.
```

### Hỏi "đang làm gì":

```
你在做什么（呢）？    Nǐ zài zuò shénme (ne)?
你在干什么（呢）？    Nǐ zài gàn shénme (ne)?  (nói tự nhiên hơn)
→ 我在学中文。        Wǒ zài xué Zhōngwén.
→ 我在上课。          Wǒ zài shàng kè.  (đang học/lên lớp)
```

### Gọi điện hỏi nhau:

```
A: 喂？你在哪儿？          Wèi? Nǐ zài nǎr?
   Alo? Bạn ở đâu?

B: 我在学校呢。你呢？      Wǒ zài xuéxiào ne. Nǐ ne?
   Tôi ở trường. Còn bạn?

A: 我在家。你在做什么呢？   Wǒ zài jiā. Nǐ zài zuò shénme ne?
   Tôi ở nhà. Bạn đang làm gì?

B: 我正在上课呢，等一下。   Wǒ zhèngzài shàng kè ne, děng yíxià.
   Tôi đang học, chờ chút nhé.
```

> 💡 **Dev tip:** `正在` giống `Thread.State.RUNNING` — process đang chạy ngay lúc này. `在` giống `isActive()` — check trạng thái. `呢` giống ` // currently running` — comment nhấn mạnh.

B: 四个半小时。早上复习，下午练习，晚上做作业。

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 校 xiào (trường) — 10 nét
```
木(cây) + 交(giao) → "nơi giao tiếp dưới cây" = TRƯỜNG
学校 = trường học
Tập viết: 校 校 校 校 校 校 校 校 校 校
```

#### 教 jiāo/jiào (dạy) — 11 nét
```
孝(hiếu) + 攵(đánh/gõ) → "truyền kiến thức" = DẠY
教室 = phòng học, 教师 = giáo viên
Tập viết: 教 教 教 教 教 教 教 教 教 教
```

#### 试 shì (thử/thi) — 8 nét
```
讠(lời) + 式(kiểu) → "kiểm tra bằng lời" = THI
考试 = thi, 试试 = thử xem
Tập viết: 试 试 试 试 试 试 试 试 试 试
```

#### 复 fù (lại/phục) — 9 nét
```
复习 = ôn tập (学lại), 复杂 = phức tạp
Tập viết: 复 复 复 复 复 复 复 复 复 复
```

#### 习 xí (tập/luyện) — 3 nét
```
羽(cánh) giản thể → "chim tập bay" = LUYỆN TẬP
学习  = học tập, 练习 = luyện tập, 复习 = ôn tập
Tập viết: 习 习 习 习 习 习 习 习 习 习
```

#### 业 yè (nghiệp/nghề) — 5 nét
```
作业 = bài tập, 工业 = công nghiệp, 企业 = doanh nghiệp
Tập viết: 业 业 业 业 业 业 业 业 业 业
```

#### 紧 jǐn (chặt/gấp) — 10 nét
```
紧张 = căng thẳng/lo lắng, 紧急 = khẩn cấp
Tập viết: 紧 紧 紧 紧 紧 紧 紧 紧 紧 紧
```

#### 经 jīng (kinh/qua) — 8 nét
```
纟(sợi tơ) + 巠 → "sợi dọc" = KINH/QUA
已经 = đã rồi, 经常 = thường xuyên, 经理 = giám đốc
Tập viết: 经 经 经 经 经 经 经 经 经 经
```
