## 📐 Phần 1: Ngữ Pháp — 能/可以/会 (30 phút)

### Ba cách nói "có thể":

```
┌─────────────────────────────────────────────────────────┐
│  能 néng     = có thể (KHẢ NĂNG thể chất/điều kiện)    │
│  可以 kěyǐ   = có thể (ĐƯỢC PHÉP/cho phép)             │
│  会 huì      = biết/sẽ (BIẾT LÀM/đã học được)          │
└─────────────────────────────────────────────────────────┘
```

### 1️⃣ 能 néng = CÓ THỂ (khả năng/điều kiện)

```
我能跑十公里。          Wǒ néng pǎo shí gōnglǐ.
Tôi CÓ THỂ chạy 10km. (thể lực cho phép)

今天太忙了，我不能来。  Jīntiān tài máng le, wǒ bù néng lái.
Hôm nay bận quá, tôi KHÔNG THỂ đến. (điều kiện không cho phép)

你能帮我一下吗？        Nǐ néng bāng wǒ yíxià ma?
Bạn CÓ THỂ giúp tôi một chút không?
```

### 2️⃣ 可以 kěyǐ = ĐƯỢC PHÉP (cho phép/đồng ý)

```
这里可以拍照吗？        Zhèlǐ kěyǐ pāi zhào ma?
Ở đây ĐƯỢC chụp ảnh không? (hỏi phép)

可以！/ 不可以。         Kěyǐ! / Bù kěyǐ.
Được! / Không được.

老师，我可以去厕所吗？  Lǎoshī, wǒ kěyǐ qù cèsuǒ ma?
Thầy ơi, em ĐƯỢC đi WC không?
```

### 3️⃣ 会 huì = BIẾT LÀM / SẼ (đã học được)

```
我会说中文。            Wǒ huì shuō Zhōngwén.
Tôi BIẾT nói tiếng Trung. (đã học)

你会写代码吗？          Nǐ huì xiě dàimǎ ma?
Bạn BIẾT viết code không? (kỹ năng)

明天会下雨。            Míngtiān huì xià yǔ.
Ngày mai SẼ mưa. (dự đoán)

他不会做饭。            Tā bú huì zuò fàn.
Anh ấy KHÔNG BIẾT nấu cơm.
```

### So sánh 3 cái trong 1 tình huống:

```
🏊 Bơi:
  我会游泳。        Tôi BIẾT bơi. (đã học)
  我能游一公里。    Tôi CÓ THỂ bơi 1km. (sức khỏe đủ)
  这里可以游泳。    Ở đây ĐƯỢC bơi. (cho phép)

💻 Code:
  我会写Python。     Tôi BIẾT viết Python. (đã học)
  今天能写完代码。   Hôm nay CÓ THỂ viết xong code. (có thời gian)
  公司可以用GitHub。  Cty ĐƯỢC dùng GitHub. (cho phép)
```

> 💡 **Dev tip:**
> - `会` = `hasSkill("swimming")` — check kỹ năng
> - `能` = `canPerform("swimming", currentHealth)` — check điều kiện runtime
> - `可以` = `isAllowed("swimming", location)` — check permission

## 📚 Phần 2: Giao Tiếp & Điện Thoại (20 phút)

### Gọi điện thoại:

```
喂？                     Wèi?         A-lô?
你好，请问是谁？         Xin chào, ai đó ạ?
我是小王。               Tôi là Tiểu Vương.
请等一下。               Xin chờ một chút.
他不在。                 Anh ấy không có ở đây.
你能不能给他打电话？     Bạn có thể gọi cho anh ấy không?
好的，我等一下再打。     OK, lát nữa tôi gọi lại.
```

### Các phương thức liên lạc:

```
打电话     dǎ diànhuà     gọi điện thoại
发短信     fā duǎnxìn     gửi tin nhắn SMS
发微信     fā wēixìn      gửi WeChat (= Zalo TQ)
发邮件     fā yóujiàn     gửi email
回         huí             hồi âm/trả lời
联系       liánxì          liên lạc
```

### Mẫu hội thoại điện thoại:

```
A: 喂？你好！
B: 你好，请问你是小李吗？
A: 是的，你是谁？
B: 我是小王。你现在方便说话吗？
A: 可以，你说吧。
B: 明天你能不能来公司开会？
A: 明天几点？
B: 下午两点。
A: 好的，没问题，我会去的。
B: 太好了！那明天见！
A: 明天见！
```

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 能 néng (năng/có thể) — 10 nét
```
能 = "con gấu đứng dậy" → CÓ KHẢ NĂNG
能力 = năng lực, 不能 = không thể
Tập viết: 能 能 能 能 能 能 能 能 能 能
```

#### 会 huì (hội/biết) — 6 nét
```
人(người) + 云(mây) giản thể → "hiểu biết" = BIẾT/SẼ
会议 = cuộc họp, 开会 = họp, 学会 = học được
Tập viết: 会 会 会 会 会 会 会 会 会 会
```

#### 话 huà (lời/chuyện) — 8 nét
```
讠(lời) + 舌(lưỡi) → "lưỡi nói lời" = LỜI NÓI
说话 = nói chuyện, 电话 = điện thoại
Tập viết: 话 话 话 话 话 话 话 话 话 话
```

#### 帮 bāng (giúp) — 9 nét
```
帮忙 = giúp đỡ, 帮我 = giúp tôi
Tập viết: 帮 帮 帮 帮 帮 帮 帮 帮 帮 帮
```

#### 等 děng (đợi/chờ) — 12 nét
```
竹(tre) + 寺(chùa) → "đợi ở chùa" = ĐỢI
等一下 = chờ chút, 等等 = chờ chờ
Tập viết: 等 等 等 等 等 等 等 等 等 等
```

#### 接 jiē (nhận/tiếp) — 11 nét
```
扌(tay) + 妾 → "tay tiếp nhận" = NHẬN/TIẾP
接电话 = nghe ĐT, 接受 = tiếp nhận
Tập viết: 接 接 接 接 接 接 接 接 接 接
```

#### 题 tí (đề) — 15 nét
```
是(là) + 页(trang) → "đề mục trên trang" = ĐỀ
问题 = vấn đề, 题目 = đề bài
Tập viết: 题 题 题 题 题 题 题 题 题 题
```

#### 便 biàn/pián (tiện/rẻ) — 9 nét
```
亻(người) + 更(thêm) → "thuận tiện cho người"
方便 = thuận tiện (biàn), 便宜 = rẻ (pián)
Tập viết: 便 便 便 便 便 便 便 便 便 便
```
