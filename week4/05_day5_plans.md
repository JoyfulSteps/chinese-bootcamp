## 📐 Phần 1: Ngữ Pháp — 被 (Bị Động) (25 phút)

### Pattern: S + 被 + (người gây) + V + kết quả

```
被 bèi = bị/được (câu bị động)

Chủ động:  他吃了我的蛋糕。     Anh ấy ĐÃ ăn bánh tôi.
Bị động:   我的蛋糕被他吃了。   Bánh tôi BỊ anh ấy ăn rồi.
```

### Ví dụ:

```
✅ 被 + người gây + V:
  我的手机被弟弟拿走了。
  ĐT tôi BỊ em trai lấy đi rồi.

  这本书被他借走了。
  Cuốn sách này BỊ anh ấy mượn rồi.

  他被老师批评了。
  Anh ấy BỊ thầy phê bình.

✅ 被 + V (không nói ai gây):
  我的自行车被偷了。
  Xe đạp tôi BỊ trộm rồi. (không biết ai)

  文件被删了。
  File BỊ xóa rồi.
```

### Trong đời dev:

```
代码被删了！         Code bị xóa rồi!
Bug被修好了。        Bug đã được fix rồi.
服务器被黑客攻击了。 Server bị hacker tấn công.
数据被覆盖了。       Data bị ghi đè rồi.
文件被误删了。       File bị xóa nhầm.
```

> 💡 **Dev tip:** `被` = `passive voice` = event listener: "something was DONE TO this object". `代码被删了` = `repo.on('delete', file)`. Object nhận hành động lên trước!

## 📐 Phần 2: Ngữ Pháp — 是...的 (Nhấn Mạnh) (25 phút)

### Pattern: S + 是 + (thời gian/cách/nơi) + V + 的

```
是...的 dùng để NHẤN MẠNH chi tiết (khi nào? ở đâu? bằng cách nào?)
→ Hành động ĐÃ XẢY RA, bạn muốn hỏi/nói về CHI TIẾT

⚠️ Chỉ dùng cho hành động ĐÃ XẢY RA!
```

### Ví dụ:

```
🕐 Nhấn mạnh THỜI GIAN (khi nào?):
  你是什么时候来中国的？    Bạn ĐẾN TQ KHI NÀO?
  我是去年来的。            Tôi đến NĂM NGOÁI.

📍 Nhấn mạnh NƠI CHỐN (ở đâu?):
  你是在哪儿学的中文？      Bạn học TQ Ở ĐÂU?
  我是在越南学的。          Tôi học Ở VIỆT NAM.

🔧 Nhấn mạnh CÁCH THỨC (bằng cách nào?):
  你是怎么来的？            Bạn đến BẰNG CÁCH NÀO?
  我是坐飞机来的。          Tôi đến BẰNG MÁY BAY.

👤 Nhấn mạnh NGƯỜI LÀM (ai?):
  这个是谁写的？            Cái này AI viết?
  这个是我写的。            Cái này TÔI viết.
```

### Phân biệt 了 vs 是...的:

```
我去年来了中国。        Năm ngoái tôi đến TQ. (nói sự kiện)
我是去年来的。          Tôi đến là NĂM NGOÁI. (nhấn mạnh: khi nào)

他学了中文。            Anh ấy đã học TQ. (nói sự kiện)
他是在北京学的中文。    Anh ấy học TQ Ở BẮC KINH. (nhấn mạnh: ở đâu)
```

> 💡 **Dev tip:** `是...的` giống `console.assert()` — nhấn mạnh một assertion: "Tôi khẳng định rằng thời gian là NĂM NGOÁI". Hoặc giống `highlight` trong search result — highlight phần quan trọng.

## 📚 Phần 3: Từ Vựng Kế Hoạch & Tương Lai (15 phút)

```
打算     dǎsuan       dự định/kế hoạch
计划     jìhuà        kế hoạch (formal)
准备     zhǔnbèi      chuẩn bị
希望     xīwàng       hy vọng
将来     jiānglái     tương lai
以后     yǐhòu        sau này/sau đó
```

### Nói về kế hoạch:

```
你以后打算做什么？        Sau này bạn dự định làm gì?
我打算去中国工作。        Tôi dự định đi TQ làm việc.
我在准备HSK2考试。       Tôi đang chuẩn bị thi HSK 2.
我希望明年能去中国。      Tôi hy vọng năm sau đi được TQ.
将来我想当一个好程序员。  Tương lai tôi muốn làm dev giỏi.
```

```
A: 你学中文学了多长时间了？
   Nǐ xué Zhōngwén xué le duō cháng shíjiān le?

B: 已经学了快四个星期了。
   Yǐjīng xué le kuài sì ge xīngqī le.

A: 你是怎么学的？
   Nǐ shì zěnme xué de?

B: 我是用AI学的。每天学四个半小时。
   Wǒ shì yòng AI xué de. Měi tiān xué sì ge bàn xiǎoshí.

A: 进步怎么样？

B: 还不错。已经学了快三百个词了。但是听力被我忽视了，

   Hái búcuò. Yǐjīng xué le kuài sān bǎi ge cí le. Dànshì tīnglì bèi wǒ
   hūshì le, xūyào duō liànxí.

A: 你以后打算怎么做？
   Nǐ yǐhòu dǎsuan zěnme zuò?

B: 我打算先通过HSK2考试，然后学技术中文。
   我希望以后能看懂中国的技术博客。
   Wǒ dǎsuan xiān tōngguò HSK2 kǎoshì, ránhòu xué jìshù Zhōngwén.
   Wǒ xīwàng yǐhòu néng kàn dǒng Zhōngguó de jìshù bókè.

A: 太厉害了！加油，你一定会成功的！
   Tài lìhai le! Jiāyóu, nǐ yídìng huì chénggōng de!

B: 谢谢！我会继续努力的。
   Xièxie! Wǒ huì jìxù nǔlì de.
```

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 被 bèi (bị/được) — 10 nét
```
衤(áo) + 皮(da) → "áo bị rách" = BỊ
被子 = chăn (mền), 被 + V = bị V
Tập viết: 被 被 被 被 被 被 被 被 被 被
```

#### 算 suàn (tính/đếm) — 14 nét
```
竹(tre) + 目(mắt) + 廾(hai tay) → "tính toán bằng bàn tính tre"
打算 = dự định, 计算 = tính toán
Tập viết: 算 算 算 算 算 算 算 算 算 算
```

#### 备 bèi (chuẩn bị) — 8 nét
```
备 giản thể của 備
准备 = chuẩn bị, 备份 = backup (IT!)
Tập viết: 备 备 备 备 备 备 备 备 备 备
```

#### 望 wàng (vọng/mong) — 11 nét
```
亡(mất) + 月(trăng) + 王(vua) → "vua nhìn trăng mất" = MONG
希望 = hy vọng, 失望 = thất vọng
Tập viết: 望 望 望 望 望 望 望 望 望 望
```

#### 决 jué (quyết) — 6 nét
```
冫(nước đá) + 夬 → "cắt đứt" = QUYẾT
决定 = quyết định, 解决 = giải quyết
Tập viết: 决 决 决 决 决 决 决 决 决 决
```

#### 定 dìng (định) — 8 nét
```
宀(nhà) + 正(ngay) → "ổn định trong nhà" = ĐỊNH
决定 = quyết định, 一定 = chắc chắn
Tập viết: 定 定 定 定 定 定 定 定 定 定
```

#### 功 gōng (công) — 5 nét
```
工(thợ) + 力(sức) → "bỏ sức lao động" = CÔNG
成功 = thành công, 功能 = chức năng (IT!)
Tập viết: 功 功 功 功 功 功 功 功 功 功
```

#### 续 xù (tiếp) — 11 nét
```
纟(sợi tơ) + 卖(bán) giản thể → "sợi nối tiếp" = TIẾP TỤC
继续 = tiếp tục, 连续 = liên tục
Tập viết: 续 续 续 续 续 续 续 续 续 续
```
