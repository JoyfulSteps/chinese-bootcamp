## Bài Tập 1: 了 vs 过 — Phân Biệt (12 phút)

### 1.1 Chọn 了 hoặc 过:

| # | Câu | 了 hay 过？ | Đáp án |
|---|-----|-----------|--------|
| 1 | 我昨天去___北京。 | | <details><summary>🔑</summary>了 (hành động cụ thể: hôm qua)</details> |
| 2 | 我去___中国，去了两次。 | | <details><summary>🔑</summary>过 (trải nghiệm: đã từng)</details> |
| 3 | 他看___这个电影吗？ | | <details><summary>🔑</summary>过 (hỏi trải nghiệm: từng xem chưa)</details> |
| 4 | 我今天吃___三碗饭。 | | <details><summary>🔑</summary>了 (hành động cụ thể: hôm nay)</details> |
| 5 | 你吃___中国菜吗？ | | <details><summary>🔑</summary>过 (hỏi trải nghiệm: từng ăn chưa)</details> |
| 6 | 他上个月来___越南。 | | <details><summary>🔑</summary>了 (hành động cụ thể: tháng trước)</details> |

### 1.2 Dịch sang tiếng Trung (chọn đúng 了/过):

| # | Tiếng Việt | Đáp án |
|---|-----------|--------|
| 1 | Tôi đã từng ăn sủi cảo. | <details><summary>🔑</summary>我吃过饺子。</details> |
| 2 | Tôi hôm qua đã ăn sủi cảo. | <details><summary>🔑</summary>我昨天吃了饺子。</details> |
| 3 | Bạn từng đi Bắc Kinh chưa? | <details><summary>🔑</summary>你去过北京吗？</details> |
| 4 | Tôi chưa từng thấy tuyết. | <details><summary>🔑</summary>我没看过雪。</details> |

## Bài Tập 4: Điền Vào Chỗ Trống (10 phút)

```
Dùng: 过 / 了 / 没 / 春天 / 冬天 / 季节 / 暖和 / 变 / 凉快
```

| # | Câu | Đáp án |
|---|-----|--------|
| 1 | 你去___中国吗？— ___，我没去___。 | <details><summary>🔑</summary>你去**过**中国吗？— **没**，我没去**过**。</details> |
| 2 | ___的时候很___，花都开___。 | <details><summary>🔑</summary>**春天**的时候很**暖和**，花都开**了**。</details> |
| 3 | 天气___了，从热___冷___。 | <details><summary>🔑</summary>天气**变**了，从热**变**冷**了**。</details> |
| 4 | 你喜欢什么___？— 我喜欢秋天，很___。 | <details><summary>🔑</summary>你喜欢什么**季节**？— 我喜欢秋天，很**凉快**。</details> |

## Bài Tập 6: Đọc Hiểu Nâng Cao (10 phút)

```
我是越南人，我没去过中国。但是我的同事去过
两次。他说北京的冬天很冷，零下十度！他看过
雪，觉得非常美。

我最想去中国过春节。因为我听说春节的时候
大家都回家，吃饺子，很热闹。越南也有春节，
我们叫"Tết"。越南人过春节的时候吃粽子，
不吃饺子。

我觉得中国的春天最好。春天暖和，
不冷不热，可以去很多地方玩。
```

|---|------|--------|
| 1 | 作者去过中国吗？ | <details><summary>🔑</summary>没去过</details> |
| 2 | 他的同事去过几次？ | <details><summary>🔑</summary>两次</details> |
| 3 | 北京冬天多少度？ | <details><summary>🔑</summary>零下十度</details> |
| 4 | 作者最想去TQ做什么？ | <details><summary>🔑</summary>过春节</details> |
| 5 | 越南人过春节吃什么？ | <details><summary>🔑</summary>吃粽子</details> |
| 6 | 作者觉得中国哪个季节最好？ | <details><summary>🔑</summary>春天最好</details> |
