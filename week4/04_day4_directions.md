## 📐 Phần 1: Ngữ Pháp — 把 (Cấu Trúc Xử Lý) (30 phút)

### Pattern: S + 把 + O + V + kết quả

```
把 bǎ = "nắm lấy" → dùng để nhấn mạnh XỬ LÝ đối tượng

Cấu trúc bình thường:  S + V + O
  我关了门。              Tôi đã đóng cửa.

Cấu trúc 把:            S + 把 + O + V + kết quả  
  我把门关了。            Tôi ĐÃ đóng cửa. (nhấn mạnh: cửa → đóng)
```

### Ví dụ chi tiết:

```
✅ 把 + V + 了/好/完/上/下/到...:
  请把门关上。        Qǐng bǎ mén guān shang.
  Hãy đóng cửa lại. (把门 = nắm lấy cửa → 关上 = đóng lại)

  把书放在桌子上。    Bǎ shū fàng zài zhuōzi shang.
  Để sách trên bàn.  (把书 = nắm sách → 放在 = đặt tại)

  我把作业做完了。    Wǒ bǎ zuòyè zuò wán le.
  Tôi đã làm xong bài tập.

  请把这个文件发给我。Qǐng bǎ zhège wénjiàn fā gěi wǒ.
  Xin gửi file này cho tôi.
```

### ⚠️ Quy tắc quan trọng:

```
1. Sau 把, phải có KẾT QUẢ (không thể chỉ có V):
   ❌ 我把门关。        (SAI — thiếu kết quả)
   ✅ 我把门关上。      (ĐÚNG — có 上 = kết quả)
   ✅ 我把门关了。      (ĐÚNG — có 了 = hoàn thành)

2. Phủ định: 没/不 đặt TRƯỚC 把:
   ✅ 我没把门关上。    Tôi chưa đóng cửa.
   ✅ 别把它扔了。      Đừng vứt nó đi.

3. 把 chỉ dùng khi O cụ thể (đã biết):
   ❌ 我把一本书看了。  (SAI — "một cuốn sách" quá chung)
   ✅ 我把那本书看了。  (ĐÚNG — "cuốn sách đó" cụ thể)
```

### Trong đời dev:

```
请把代码提交了。       Hãy commit code đi.
把bug修好了。          Đã fix bug rồi.
我把文件删了。         Tôi đã xóa file rồi.
把窗口关了。           Đóng window lại.
请把文档发给我。       Gửi docs cho tôi nhé.
```

> 💡 **Dev tip:** `把` = `pipe operator` hoặc `process()`: bạn "nắm lấy" object → xử lý → kết quả. `把门关上` = `door.close()`. `把代码提交了` = `code.commit()`. Luôn phải có method (kết quả), không thể chỉ có object!

## 📚 Phần 2: Hướng & Vị Trí (20 phút)

### Bốn hướng:

| Hán tự | Pinyin | Nghĩa |
|--------|--------|-------|
| **东** | dōng | đông |
| **西** | xī | tây |
| **南** | nán | nam |
| **北** | běi | bắc |

```
东西南北 = bốn hướng (đông tây nam bắc)
东西 dōngxi = đồ vật (thú vị: "đông + tây" = "thứ/đồ")
```

### Chỉ đường:

```
往前走     wǎng qián zǒu    đi thẳng (về phía trước)
往左拐     wǎng zuǒ guǎi    rẽ trái
往右拐     wǎng yòu guǎi    rẽ phải
一直走     yìzhí zǒu         đi thẳng
在左边     zài zuǒbiān       ở bên trái
在右边     zài yòubiān       ở bên phải
在对面     zài duìmiàn       ở đối diện
在旁边     zài pángbiān      ở bên cạnh
```

### Hỏi đường:

```
请问，银行在哪儿？     Xin hỏi, ngân hàng ở đâu?
请问，怎么去火车站？   Xin hỏi, đi ga tàu bằng cách nào?
从这里到那里远不远？   Từ đây đến đó xa không?

→ 往前走，到路口往右拐，银行在左边。
  Đi thẳng, đến ngã tư rẽ phải, NH ở bên trái.
```

### Bài Đọc 4 — 找银行 (130 chữ)

```
今天我要去银行。我不知道银行在哪儿，
所以我问了一个人。

"请问，这里附近有银行吗？"
"有。你从这里往前走，到第一个路口
往右拐。然后一直走，走五分钟。
银行在路的左边，在一家饭店的旁边。"
"谢谢你！"

我按照他说的走了。到了路口，我往右拐了。
走了五分钟，看到了那家饭店。
但是银行在哪儿呢？

我找了半天，原来银行在饭店的对面，
不是旁边！他说错了。不过没关系，
我还是找到了。我把钱存了，然后买了
一些东西回家了。
```

## ✏️ Phần 6: 8 Hán Tự Mới (20 phút)

#### 东 dōng (đông) — 5 nét
```
东 giản thể của 東
东方 = phương đông, 东西 = đồ vật
Tập viết: 东 东 东 东 东 东 东 东 东 东
```

#### 南 nán (nam) — 9 nét
```
越南 = Việt Nam (phía nam của Việt), 南方 = phương nam
Tập viết: 南 南 南 南 南 南 南 南 南 南
```

#### 北 běi (bắc) — 5 nét
```
北京 = kinh đô phương bắc, 北方 = phương bắc
Tập viết: 北 北 北 北 北 北 北 北 北 北
```

#### 左 zuǒ (trái) — 5 nét
```
工(thợ) + 手(tay ẩn) → "tay trái" = TRÁI
往左拐 = rẽ trái
Tập viết: 左 左 左 左 左 左 左 左 左 左
```

#### 右 yòu (phải) — 5 nét
```
口(miệng) + 手(tay ẩn) → "tay phải" = PHẢI
在右边 = ở bên phải
Tập viết: 右 右 右 右 右 右 右 右 右 右
```

#### 放 fàng (đặt/để/thả) — 8 nét
```
方(vuông) + 攵(đánh) → "đặt vào vị trí" = ĐẶT/ĐỂ
把书放好 = để sách gọn gàng
Tập viết: 放 放 放 放 放 放 放 放 放 放
```

#### 路 lù (đường) — 13 nét
```
足(chân) + 各(mỗi) → "chân đi mỗi nơi" = ĐƯỜNG
马路 = đường lớn, 路口 = ngã tư
Tập viết: 路 路 路 路 路 路 路 路 路 路
```

#### 找 zhǎo (tìm) — 7 nét
```
扌(tay) + 戈(giáo) → "tay cầm giáo tìm kiếm" = TÌM
找到 = tìm thấy, 找不到 = không tìm thấy
Tập viết: 找 找 找 找 找 找 找 找 找 找
```
