# 🎴 Hướng Dẫn Anki — Chiến Lược Flashcards

> Anki là công cụ **BẮT BUỘC** trong khóa học này.
> Tài liệu này trả lời câu hỏi: "Dùng deck có sẵn hay tự tạo?"

---

## 💡 Kết Luận Trước: KẾT HỢP CẢ HAI

```
DECK CÓ SẴN (HSK)     →  Học TRƯỚC mỗi ngày (15–20 phút sáng)
                           Giúp preview từ sắp học + ôn từ cũ
                           
DECK TỰ TẠO (Custom)  →  Thêm VÀO sau mỗi bài học (10–15 phút tối)
                           Ghi lại từ/câu khó mà deck có sẵn không cover
                           Personalized cho nhu cầu tech + bạn
```

### Tại sao kết hợp?

| | Deck có sẵn | Deck tự tạo |
|---|---|---|
| ✅ **Ưu** | Đầy đủ, có audio, ảnh, theo chuẩn HSK | Cá nhân hóa, liên quan đến bạn |
| ✅ **Ưu** | Tiết kiệm thời gian tạo card | Nhớ tốt hơn vì tự viết (generation effect) |
| ❌ **Nhược** | Có thể chứa từ bạn chưa cần | Mất thời gian tạo |
| ❌ **Nhược** | Thứ tự có thể khác lộ trình | Có thể thiếu audio |

## 📦 Deck Có Sẵn Khuyên Dùng

### Tải từ AnkiWeb (ankiweb.net/shared/decks)

| Deck | Tìm kiếm | Ghi chú |
|------|----------|---------|
| **HSK 1 Complete** | "HSK 1" hoặc "HSK1 Vocab" | Chọn deck có ≥ 4.5⭐, có audio |
| **HSK 2 Complete** | "HSK 2" | Tải sẵn, activate từ tuần 3 |
| **HSK 3 Complete** | "HSK 3" | Tải sẵn, activate từ tuần 5 |
| **Hanzi Radicals** | "Chinese radicals" | Học 214 bộ thủ (optional) |

### Cách dùng deck có sẵn

```
Settings khuyên dùng cho HSK deck:

New cards/day:     10–15 (không quá 20!)
Maximum reviews:   100 (giới hạn để không overwhelm)
Learning steps:    1m 10m (2 bước)
Graduating:        1d
Easy interval:     4d
Maximum interval:  30d (vì bạn chỉ có 2 tháng)

⚠️ QUAN TRỌNG: Set "Maximum interval" = 30 ngày
   Vì khóa học chỉ 56 ngày, không cần review sau 30 ngày.
```

## ✏️ Deck Tự Tạo — Hướng Dẫn

### Khi nào thêm card tự tạo?

```
✅ Thêm khi:
  - Gặp từ mới trong bài tập mà deck có sẵn chưa có
  - Từ tech mà HSK deck không cover (编程, 代码, 框架...)
  - Câu ngữ pháp pattern muốn nhớ
  - Câu ví dụ hay từ bài đọc
  - Các cặp từ dễ nhầm (在/再, 做/作, 的/得/地)

❌ KHÔNG thêm khi:
  - Từ đã có trong deck có sẵn (tránh duplicate)
  - Quá nhiều card 1 ngày (max thêm 5-8 card/ngày)
  - Từ quá khó/hiếm chưa cần lúc này
```

### Note Types cho deck tự tạo

#### Type 1: Từ Vựng (Vocabulary)

```
┌─────────────────────────────────┐
│ FRONT (Mặt trước)               │
│                                  │
│  汉字:  编程                     │
│  Pinyin: biānchéng              │
│  🔊 [Audio nếu có]              │
│                                  │
│  Gõ nghĩa: [_______________]   │
├─────────────────────────────────┤
│ BACK (Mặt sau)                   │
│                                  │
│  ✅ lập trình (programming)     │
│                                  │
│  Câu ví dụ:                     │
│  我喜欢编程。                    │
│  Wǒ xǐhuān biānchéng.         │
│  (Tôi thích lập trình)          │
│                                  │
│  Bộ thủ: 纟(sợi) + 呈(trình)   │
│  Nguồn: Day 29 - Tech Vocab     │
└─────────────────────────────────┘
```

#### Type 2: Ngữ Pháp (Grammar Pattern)

```
┌─────────────────────────────────┐
│ FRONT                            │
│                                  │
│  Pattern: 因为___所以___         │
│  Hint: vì...nên...              │
│                                  │
│  Hoàn thành câu:                │
│  ___有bug，___要修复。            │
│  [_______________]              │
├─────────────────────────────────┤
│ BACK                             │
│                                  │
│  因为有bug，所以要修复。          │
│  Yīnwèi yǒu bug, suǒyǐ yào   │
│  xiūfù.                         │
│  (Vì có bug nên phải fix)       │
│                                  │
│  Nguồn: Day 19 - Grammar         │
└─────────────────────────────────┘
```

#### Type 3: Hán Tự (Character Recognition)

```
┌─────────────────────────────────┐
│ FRONT                            │
│                                  │
│  Nhận biết chữ:  学              │
│  [Gõ pinyin + nghĩa]            │
├─────────────────────────────────┤
│ BACK                             │
│                                  │
│  Pinyin: xué                    │
│  Nghĩa: học (study/learn)       │
│  Bộ thủ: 子 (con) + ⺍ (tay)   │
│  Nét: 8 nét                     │
│  Từ liên quan: 学生, 学习, 大学  │
└─────────────────────────────────┘
```

## 💡 Tips Pro

1. **Đừng skip Anki** — Ngay cả ngày bận, cũng review ít nhất 10 phút
2. **Suspend cards quá dễ** — Nếu 1 từ bạn luôn đúng ≥ 5 lần liên tiếp, suspend nó
3. **Tag cards** — Tag theo tuần: `week1`, `week2`... và theo loại: `vocab`, `grammar`, `hanzi`, `tech`
4. **Ankidroid/AnkiMobile** — Cài trên điện thoại, review lúc đi xe buýt, chờ đợi
5. **Không quá nhiều new cards** — 10–15 new/day cho deck có sẵn + 3–5 new/day tự tạo = 13–20 total. MAX 20.
