🏋️ Luyện tập ghép âm **yi, wu, yu**:

📍 Phân loại: Bài luyện tập

✅ NỘI DUNG LUYỆN:

- **yi** = i đứng đầu → 一 yī (một), 以 yǐ (lấy)
- **wu** = u đứng đầu → 五 wǔ (năm), 无 wú (không)
- **yu** = ü đứng đầu → 鱼 yú (cá), 雨 yǔ (mưa)

📝 QUY TẮC: Khi i/u/ü đứng đầu âm tiết:

- **i** → thêm y → **yi**
- **u** → thêm w → **wu**
- **ü** → đổi thành **yu** (bỏ 2 chấm)

🧪 Luyện: yī-yí-yǐ-yì / wū-wú-wǔ-wù / yū-yú-yǔ-yù (4 thanh cho mỗi âm!)

🧪 TỔNG KẾT -n vs -ng:

| -n (trước) | -ng (sau) |
|:----------:|:---------:|
| an ≈ "an" | ang ≈ "ang" |
| en ≈ "ân" | eng ≈ "ơng" |
| in ≈ "in" | ing ≈ "inh" |
| un ≈ "uân" | ong ≈ "ung" |

🧠 TRICK: -n = lưỡi chạm RĂNG (trước). -ng = lưỡi chạm VÒM (sau)!
